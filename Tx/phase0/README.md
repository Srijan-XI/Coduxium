# Phase 0 Artifacts

This folder contains the implementation outputs for Phase 0.

## Files
- PHASE_0_DEFINITION_OF_DONE.md
- CONTENT_SCHEMA_CHECKLIST.md
- templates/

## Templates
The templates folder contains one master template for each required domain:
- Languages
- Frameworks
- DevOps/Cloud
- Data/DB
- Security
- Mobile
- Web3
- Game frameworks
- Utilities/OS

## Usage
1. Copy the domain template.
2. Replace placeholder fields.
3. Adjust relative paths by page depth.
4. Validate against CONTENT_SCHEMA_CHECKLIST.md.
5. Validate against PHASE_0_DEFINITION_OF_DONE.md.
