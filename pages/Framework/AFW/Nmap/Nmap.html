Nmap

Nmap is a powerful network discovery and security auditing tool used to identify hosts, open ports, services, and vulnerabilities.

Overview

Network Security Port Scanning Diagnostics

Nmap provides a wide range of scanning techniques and options, making it a staple for network administrators and security professionals to assess exposure and validate configurations.

Nmap helps security professionals and developers understand network exposure, validate firewall rules, and detect misconfigurations.

Common Use Cases

  • Network inventory and host discovery
  • Security auditing and vulnerability scanning
  • Firewall and segmentation validation

Scan Types

  • -sS: SYN scan (stealthy and fast)
  • -sT: TCP connect scan
  • -sU: UDP scan
  • -sn: Host discovery only

Important Flags

  • -p → Select ports
  • -A → Aggressive scan
  • -O → OS detection
  • -T0 - T5 → Speed control
  • --open → Show only open ports
  • -oN → Save output

Basic Workflow

  1. Discover hosts → nmap -sn 192.168.1.0/24
  2. Scan ports → nmap -sS 192.168.1.10
  3. Detect services → nmap -sV 192.168.1.10
  4. Run scripts → nmap --script vuln 192.168.1.10

Common Commands

nmap 192.168.1.0/24
nmap -sV 10.0.0.15
nmap -A target.com

Understanding Output

  • PORT: Port number
  • STATE: open / closed / filtered
  • SERVICE: Running service
  • VERSION: Service version

Example Output

80/tcp open http Apache 2.4.41
