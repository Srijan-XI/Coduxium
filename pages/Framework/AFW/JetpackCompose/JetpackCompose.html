Jetpack Compose

Jetpack Compose is Android's declarative UI toolkit for building native interfaces with Kotlin and unidirectional state patterns.

Overview

Android Native Kotlin Declarative Material 3

Compose improves UI productivity by replacing verbose XML layout management with composable functions and reactive state handling.

Key Features

  • Composable Functions - reusable UI blocks in Kotlin
  • State-driven Rendering - predictable updates with immutable models
  • Material Design - first-class support for Material 3
  • Interoperability - gradual migration from XML views
  • Tooling - previews, inspection, and performance profilers

Getting Started

// build.gradle (module)
android {
  buildFeatures {
    compose true
  }
  composeOptions {
    kotlinCompilerExtensionVersion "1.5.15"
  }
}
// MainActivity.kt
setContent {
  MaterialTheme {
    Text("Hello Compose")
  }
}
