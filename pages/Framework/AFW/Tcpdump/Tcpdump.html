tcpdump

tcpdump is a command-line packet analyzer used to capture, inspect, and troubleshoot network traffic in real time.

Overview

Network Packet Capture Diagnostics CLI Tooling

tcpdump gives operators low-level visibility into packets and protocol exchanges, making it a core troubleshooting and incident analysis utility.

Common Use Cases

  • Investigating latency, drops, and retransmissions
  • Debugging DNS, TLS, and HTTP connectivity issues
  • Capturing traffic for offline analysis in Wireshark

Getting Started

# Capture traffic on a specific interface
tcpdump -i eth0

# Filter by host and port
tcpdump -i eth0 host 10.0.0.15 and port 443
# Write capture to a file
tcpdump -i eth0 -w capture.pcap

Safety Note

Capture traffic only where authorized. Packet captures may contain sensitive data and should be handled according to security policy.
