Kivy

Kivy is an open-source Python framework for creating multitouch applications that run on Android, iOS, Linux, macOS, and Windows.

Overview

Python Cross-platform Touch UI Open Source

Kivy targets developers who want Python productivity for app interfaces and interaction-heavy experiences across desktop and mobile platforms.

Key Features

  • KV Language - declarative UI syntax for layout separation
  • Input Support - multitouch, mouse, and keyboard handling
  • Graphics Engine - OpenGL-backed rendering pipeline
  • Cross-platform Packaging - Buildozer and platform tooling
  • Widget Ecosystem - buttons, lists, screen managers, gestures
