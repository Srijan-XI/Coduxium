MetaMask

MetaMask is a widely used wallet for browser and mobile environments, enabling account management and transaction signing for dApps.

Overview

Web3 Wallet Transaction Signing dApp Identity

MetaMask acts as an injected provider in web apps and as a user-controlled signing layer for account access, chain switching, and secure transaction confirmation.

Core Flows

  • Detect provider and request account connection
  • Prompt chain/network switching where required
  • Sign messages for wallet-based authentication
  • Sign and broadcast contract transactions
  • Handle account and chain change events in UI state

Getting Started

// Basic provider check
if (window.ethereum) {
  const accounts = await window.ethereum.request({
    method: 'eth_requestAccounts'
  });
  console.log('Connected account:', accounts[0]);
}
// Listen for context changes
window.ethereum.on('accountsChanged', (accounts) => {
  console.log('Accounts changed:', accounts);
});

window.ethereum.on('chainChanged', (chainId) => {
  console.log('Chain changed:', chainId);
});

Security and UX Practices

  • Explain signature prompts before triggering wallet popups
  • Never request unnecessary permissions
  • Validate chain ID and contract addresses explicitly
  • Offer clear fallback messages when wallet is unavailable
