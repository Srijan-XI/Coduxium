Stencil

Stencil is a compiler for generating standards-based Web Components with a TypeScript-first authoring experience.

Overview

Frontend Web Components Compiler Design Systems

Stencil helps teams build reusable component libraries that work with React, Vue, Angular, and plain HTML. It compiles modern authoring patterns into standards-compliant custom elements.

Key Features

  • TypeScript-first — robust tooling and IDE integration
  • Output targets — adapters for multiple framework consumers
  • Lazy-loading — optimized runtime delivery per component
  • Testing support — unit and end-to-end component testing

Common Use Cases

  • Enterprise design systems shared by multiple product teams
  • Cross-framework UI kits with one source of truth
  • Micro-frontend platforms requiring standards-based components
  • Long-lived component libraries with minimal framework lock-in

Getting Started

npm init stencil
cd my-component-lib
npm install
npm run start
# build
npm run build

# test
npm test

Best Practices

  • Design components with clear API contracts and accessibility defaults.
  • Version components semantically and publish migration notes.
  • Generate framework wrappers only when consumers need them.
  • Automate visual and behavioral regression tests in CI.
