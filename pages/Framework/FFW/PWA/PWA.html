Progressive Web Apps (PWA)

PWAs provide app-like behavior on the web through installability, offline support, push notifications, and resilient loading.

Overview

Mobile Web Offline-first Installable Service Workers

PWA architecture helps teams deliver reliable UX across devices without requiring native app store distribution for every release.

Core Capabilities

  • Web App Manifest - app metadata, icons, and install prompts
  • Service Workers - request interception and cache control
  • Offline Patterns - cache-first, network-first, stale-while-revalidate
  • Push Notifications - browser-based engagement channel
  • Background Sync - deferred operations for poor networks

Getting Started

# Example with Vite + PWA plugin
npm create vite@latest my-pwa
cd my-pwa
npm install
npm install vite-plugin-pwa -D
npm run dev
# Validate baseline quality
npx lighthouse http://localhost:5173 --view
