📚 Introduction

What is Solid.js?

Solid.js is a declarative JavaScript library for creating user interfaces with fine-grained reactivity. Unlike React, it doesn't use a Virtual DOM - instead, it compiles your code into efficient DOM updates, resulting in exceptional performance. It uses JSX but the similarity to React ends there.

Key Features

⚡ Blazing Fast — No Virtual DOM overhead
🎯 Fine-Grained Reactivity — Surgical DOM updates
📦 Small Bundle Size — ~7KB minified + gzipped
🔧 JSX Templates — Familiar syntax
🚀 Real Reactivity — True reactive primitives
📈 Predictable — No re-renders, no hooks rules
🌐 SSR Ready — Server-side rendering support
🎨 No Runtime — Compiled to vanilla JS

Why Choose Solid.js?

  • Performance: Fastest framework in most benchmarks
  • Simple Mental Model: No complex lifecycle, just reactivity
  • React-Like DX: Familiar JSX without the gotchas
  • Bundle Size: Smaller than React or Vue
  • No Magic: Predictable, explicit behavior
  • True Reactivity: Data flows naturally
⚠️ Note: Solid.js looks like React but behaves differently. JSX is compiled, not runtime-rendered.

⚙️ Installation & Setup

Get started with Solid.js using Vite or the official templates.

Create New Project

# Create new project with Vite
npm create vite@latest my-solid-app -- --template solid

# Or with TypeScript
npm create vite@latest my-solid-app -- --template solid-ts

cd my-solid-app
npm install

Reactive Primitives - Signal

import { createSignal } from 'solid-js';

function Counter() {
  const [count, setCount] = createSignal(0);
  
  return (
    <div>
      <p>Count: {count()}</p>
      <button onClick={() => setCount(count() + 1)}>
        Increment
      </button>
    </div>
  );
}

Memos - createMemo

import { createSignal, createMemo } from 'solid-js';

function ExpensiveCalculation() {
  const [count, setCount] = createSignal(0);
  
  // Cached computed value
  const doubled = createMemo(() => {
    console.log('Computing...');
    return count() * 2;
  });
  
  return (
    <div>
      <p>Count: {count()}</p>
      <p>Doubled: {doubled()}</p>
      <button onClick={() => setCount(count() + 1)}>+</button>
    </div>
  );
}

Control Flow - Show, For

import { createSignal, Show, For } from 'solid-js';

function TodoList() {
  const [todos, setTodos] = createSignal([
    { id: 1, text: 'Learn Solid', done: false },
    { id: 2, text: 'Build app', done: false }
  ]);
  
  return (
    <div>
      <Show when={todos().length > 0} fallback={<p>No todos</p>}>
        <ul>
          <For each={todos()}>
            {(todo) => (
              <li style={{ 
                "text-decoration": todo.done ? "line-through" : "none" 
              }}>
                {todo.text}
              </li>
            )}
          </For>
        </ul>
      </Show>
    </div>
  );
}

Store (State Management)

import { createStore } from 'solid-js/store';

function TodoApp() {
  const [todos, setTodos] = createStore([
    { id: 1, text: 'Learn Solid', completed: false }
  ]);
  
  const addTodo = (text: string) => {
    setTodos([...todos, { 
      id: Date.now(), 
      text, 
      completed: false 
    }]);
  };
  
  const toggleTodo = (id: number) => {
    setTodos(
      (todo) => todo.id === id,
      'completed',
      (c) => !c
    );
  };
  
  return (
    <For each={todos}>
      {(todo) => (
        <div onClick={() => toggleTodo(todo.id)}>
          {todo.text} - {todo.completed ? '✓' : '○'}
        </div>
      )}
    </For>
  );
}

Resources (Async Data)

import { createResource } from 'solid-js';

async function fetchUser(id) {
  const response = await fetch(`/api/users/${id}`);
  return response.json();
}

function UserProfile() {
  const [userId, setUserId] = createSignal(1);
  const [user] = createResource(userId, fetchUser);
  
  return (
    <Show when={user()} fallback={<div>Loading...</div>}>
      <div>
        <h1>{user().name}</h1>
        <p>{user().email}</p>
      </div>
    </Show>
  );
}

🚀 Advanced Topics

Context API

import { createContext, useContext } from 'solid-js';

const CounterContext = createContext();

function CounterProvider(props) {
  const [count, setCount] = createSignal(0);
  
  const counter = {
    count,
    increment: () => setCount(c => c + 1),
    decrement: () => setCount(c => c - 1)
  };
  
  return (
    <CounterContext.Provider value={counter}>
      {props.children}
    </CounterContext.Provider>
  );
}

function useCounter() {
  return useContext(CounterContext);
}

function Display() {
  const { count, increment } = useCounter();
  return <button onClick={increment}>{count()}</button>;
}
