🟢 Vue.js

The progressive JavaScript framework for building modern web interfaces with an approachable core library.

Overview

Frontend Progressive Framework Reactive Component-Based

Vue.js is a progressive framework for building user interfaces. Unlike monolithic frameworks, Vue is designed from the ground up to be incrementally adoptable. The core library focuses on the view layer only, making it easy to integrate with other libraries or existing projects.

Key Features

  • Progressive Framework — Scalable from simple library to full framework
  • Reactive Data Binding — Automatic UI updates when data changes
  • Component System — Reusable and composable components
  • Single-File Components — HTML, CSS, and JavaScript in one file
  • Vue Router — Official routing solution
  • Vuex/Pinia — State management patterns
  • Virtual DOM — Efficient rendering and updates
  • Composition API — Better logic reuse and code organization

Common Use Cases

  • Single Page Applications (SPAs)
  • Progressive Web Apps (PWAs)
  • Interactive dashboards and admin panels
  • E-commerce platforms
  • Real-time applications
  • Form-heavy applications
  • Content management systems
  • Incremental adoption in existing projects

Vue Ecosystem

  • Nuxt.js — Full-featured framework with SSR/SSG
  • Vite — Official build tool with instant HMR
  • Vue Router — Official routing library
  • Pinia — Official state management (replaces Vuex)
  • VueUse — Collection of essential composition utilities
  • Vuetify/Quasar — Material Design component frameworks
  • Vue DevTools — Browser extension for debugging
  • Vitest — Testing framework

Core Concepts

Options API vs Composition API

  • Options API — Traditional, beginner-friendly approach
  • Composition API — Flexible, reusable logic composition
  • Both APIs are fully supported and interchangeable

Reactivity System

  • ref() — Create reactive references
  • reactive() — Create reactive objects
  • computed() — Computed properties
  • watch() — Watch reactive changes
  • watchEffect() — Automatic dependency tracking

Template Syntax

  • Directives: v-if, v-for, v-model, v-bind, v-on
  • Template interpolation with mustache syntax
  • Event handling and modifiers
  • Conditional rendering and list rendering

Getting Started

Create a new Vue 3 project with Vite:

npm create vue@latest my-app
cd my-app
npm install
npm run dev

Or quick start with CDN:

<script src="https://unpkg.com/vue@3"></script>
<div id="app">{{ message }}</div>
<script>
  Vue.createApp({
    data() { return { message: 'Hello Vue!' } }
  }).mount('#app')
</script>

Best Practices

  • Use Composition API for complex logic
  • Keep components small and focused
  • Leverage computed properties for derived state
  • Use Pinia for global state management
  • Implement proper prop validation
  • Use TypeScript for type safety
  • Follow Vue Style Guide conventions
  • Optimize with async components and lazy loading
