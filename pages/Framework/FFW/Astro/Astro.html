📚 Introduction

What is Astro?

Astro is an all-in-one web framework for building fast, content-focused websites. It pioneered the "Islands Architecture" pattern, allowing you to use any UI framework (React, Vue, Svelte, etc.) while shipping minimal JavaScript to the browser. By default, Astro renders everything to static HTML at build time.

Key Features

🏝️ Islands Architecture — Interactive components on demand
⚡ Zero JS by Default — Ship only what you need
🎨 Framework Agnostic — React, Vue, Svelte, Solid
📦 Content Collections — Type-safe content management
🔧 Built-in Features — Routing, layouts, assets
🌐 SSR & SSG — Server or static rendering
📈 SEO Optimized — Perfect Lighthouse scores
🚀 Fast Builds — Vite-powered development

Why Choose Astro?

  • Performance: Ships minimal JavaScript, blazing fast load times
  • Flexibility: Use any UI framework or none at all
  • Content-First: Perfect for blogs, docs, marketing sites
  • Developer Experience: File-based routing, hot reload
  • SEO Friendly: Static HTML, meta tags, sitemaps
  • Easy Migration: Bring your existing React/Vue components
⚠️ Note: Astro 4.0 introduced View Transitions API support and improved content collections.

⚙️ Installation & Setup

Get started with Astro using the official CLI.

# - Initialize git? Yes

cd my-astro-site

Content Collections

// src/content/config.ts
import { defineCollection, z } from 'astro:content';

const blogCollection = defineCollection({
  type: 'content',
  schema: z.object({
    title: z.string(),
    pubDate: z.date(),
    description: z.string(),
    author: z.string(),
    tags: z.array(z.string())
  })
});

export const collections = {
  'blog': blogCollection
};

// src/content/blog/first-post.md
