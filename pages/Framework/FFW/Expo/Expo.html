Expo

Expo is a productivity layer for React Native that helps teams build, ship, and maintain mobile apps faster with a managed workflow.

Overview

Mobile React Native Managed Workflow OTA Updates

Expo combines development tooling, cloud services, and runtime APIs so teams can focus on product features instead of native build complexity.

Key Features

  • Managed Projects - build mobile apps without full native setup
  • EAS Build - cloud builds for Android and iOS
  • EAS Update - over-the-air JavaScript updates
  • Expo SDK - camera, notifications, sensors, and device APIs
  • Dev Client - custom native modules when needed

Getting Started

npx create-expo-app@latest my-expo-app
cd my-expo-app
npm install
npx expo start
# Optional cloud build setup
npx expo login
npx eas init
