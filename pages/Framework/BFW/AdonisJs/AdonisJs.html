AdonisJS

AdonisJS is a TypeScript-first backend framework that provides a cohesive, batteries-included developer experience for full-featured web applications.

Overview

Backend TypeScript MVC Batteries Included

AdonisJS combines routing, validation, ORM, auth, and CLI tooling into a consistent framework suited for team productivity.

Key Features

  • Integrated tooling — CLI, migrations, and code generation
  • Lucid ORM — expressive data modeling for SQL backends
  • Validation and auth — common backend concerns built in
  • Strong conventions — easier onboarding and consistency

Common Use Cases

  • Monolithic business applications
  • Internal admin portals and APIs
  • Rapid MVPs with production paths
  • TypeScript-first full backend stacks

Getting Started

npm init adonisjs@latest adonis-app
cd adonis-app
node ace serve --watch
# generate controller and model
node ace make:controller Users
node ace make:model User -m

Best Practices

  • Keep domain modules focused by business capability.
  • Use validators for every external payload boundary.
  • Manage schema changes through migrations only.
  • Centralize environment configuration and secrets.
