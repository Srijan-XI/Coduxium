Elysia

Elysia is a TypeScript-first framework designed for Bun, focused on strong typing, high performance, and plugin-driven API composition.

Overview

Backend Bun Native Type Safety High Performance

Elysia targets modern runtime capabilities and is well suited for teams adopting Bun as their backend platform.

Key Features

  • End-to-end typing — route contracts with strong inference
  • Plugin ecosystem — composable capabilities and middleware
  • Bun optimization — low overhead request handling
  • Developer ergonomics — concise API definitions

Common Use Cases

  • Bun-native REST and RPC APIs
  • Latency-sensitive service endpoints
  • Type-driven backend development
  • Greenfield modern Node alternative stacks

Getting Started

bun create elysia ./elysia-app
cd elysia-app
bun install
bun run dev
// src/index.ts
import { Elysia } from 'elysia'
new Elysia()
  .get('/health', () => ({ ok: true }))
  .listen(3000)

Best Practices

  • Define typed schemas for request and response payloads.
  • Keep plugins modular by feature area.
  • Benchmark on Bun with realistic production traffic profiles.
  • Enforce structured error responses for client consistency.
