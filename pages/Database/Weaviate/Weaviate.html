Weaviate

Weaviate is an open-source vector database with hybrid search, filtering, and module-based AI integrations.

Overview

Vector DB Hybrid Search Open Source Metadata Filtering

Weaviate supports combining vector similarity with keyword and metadata filtering for production retrieval applications.

Key Features

  • Hybrid retrieval with vector and lexical ranking
  • Schema-driven objects with rich metadata filters
  • Module ecosystem for embeddings and rerankers
  • Horizontal scaling for large retrieval workloads

Getting Started

# Docker quick start
docker run -d --name weaviate -p 8080:8080 semitechnologies/weaviate:latest

# Inspect API root
curl http://localhost:8080/v1/.well-known/ready

Use Cases

  • Enterprise knowledge search with metadata constraints
  • RAG systems with hybrid relevance tuning
  • Semantic catalogs for product and media discovery
