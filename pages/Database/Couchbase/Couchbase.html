Couchbase

Couchbase is a distributed NoSQL database combining key-value performance with SQL-like querying through N1QL.

Overview

NoSQLDocumentDistributedKV + Query

Couchbase is designed for low-latency operations at scale with built-in caching, replication, and flexible data modeling.

Getting Started

# Docker quick start
