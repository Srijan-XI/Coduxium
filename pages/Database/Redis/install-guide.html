Redis — Installation Guide

Install and run Redis on Windows, macOS, and Linux. Verify with CLI and Docker.

CLI SERVICE DOCKER
🏠 Home Windows macOS Linux Docker Verify

Windows

Official Redis does not provide native Windows binaries. Use WSL2 or containers for production-like behavior.

Option — WSL2 (Ubuntu)

wsl --install -d Ubuntu
wsl
sudo apt update
sudo apt install redis-server
sudo sed -i 's/^supervised no/supervised systemd/' /etc/redis/redis.conf
sudo systemctl enable --now redis-server

macOS

brew install redis
brew services start redis
redis-cli ping

Linux

Ubuntu/Debian

sudo apt update
sudo apt install redis-server
sudo sed -i 's/^supervised no/supervised systemd/' /etc/redis/redis.conf
sudo systemctl enable --now redis-server

RHEL/CentOS/Fedora

sudo dnf install redis
sudo systemctl enable --now redis

Docker

docker run -d --name redis -p 6379:6379 redis:7-alpine

Verify

redis-cli ping
SET health "ok"
GET health
