MySQL — Installation Guide

Install MySQL on Windows, macOS, and Linux. Secure it and verify connectivity.

CLIENT SERVER SECURE
🏠 Home Windows macOS Linux Secure Verify
Use the latest MySQL 8.x. For embedded apps or lightweight usage, consider MariaDB compatibility.

Windows

Option 1 — MySQL Installer

# Download and run the official MySQL Installer
# Select: MySQL Server, MySQL Shell, MySQL Workbench

Option 2 — winget / Chocolatey

# winget
winget install Oracle.MySQL

# Chocolatey
choco install mysql

macOS

# Homebrew
brew install mysql
brew services start mysql

# Set root password (first time)
mysql_secure_installation

Linux

Ubuntu/Debian

sudo apt update
sudo apt install mysql-server
sudo systemctl enable --now mysql
sudo mysql_secure_installation

RHEL/CentOS/Fedora

sudo dnf install @mysql:8.0 mysql-server
sudo systemctl enable --now mysqld
sudo mysql_secure_installation

Secure & Create User

# Run secure installation wizard
mysql_secure_installation

# Connect as root (Linux)
sudo mysql

# Create database and user
CREATE DATABASE appdb CHARACTER SET utf8mb4;
CREATE USER 'app'@'%' IDENTIFIED BY 'StrongP@ssw0rd';
GRANT ALL PRIVILEGES ON appdb.* TO 'app'@'%';
FLUSH PRIVILEGES;

Verify

# CLI
mysql -u app -p -h 127.0.0.1 -P 3306 -D appdb -e "SELECT VERSION();"

# Create sample table
mysql -u app -p -D appdb -e "CREATE TABLE t(id INT PRIMARY KEY AUTO_INCREMENT, name VARCHAR(60));"

# Insert & query
mysql -u app -p -D appdb -e "INSERT INTO t(name) VALUES ('Ada'); SELECT * FROM t;"
