etcd

etcd is a distributed key-value store that provides strong consistency and is widely used for service discovery and cluster coordination.

Overview

KV Store Distributed Strong Consistency Coordination

etcd powers critical control planes, including Kubernetes, where reliable state coordination and leader election are required.

Key Features

  • Raft-based consensus for consistent writes
  • Watch API for change notifications
  • Leases and TTLs for ephemeral coordination data
  • Snapshot and backup support for recovery

Getting Started

# Docker quick start
docker run -d --name etcd -p 2379:2379 -p 2380:2380 \
  quay.io/coreos/etcd:v3.5.16 \
  /usr/local/bin/etcd --advertise-client-urls=http://0.0.0.0:2379 \
  --listen-client-urls=http://0.0.0.0:2379

# Set and get a key
docker exec etcd etcdctl put app/config "enabled"
docker exec etcd etcdctl get app/config
