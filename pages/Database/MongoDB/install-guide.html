MongoDB — Installation Guide

Install MongoDB Community Server on Windows, macOS, Linux, or run via Docker. Verify with mongosh.

COMMUNITY MONGOSH DOCKER
🏠 Home Windows macOS Linux Docker Verify

Windows

# winget (MongoDB Community Server)
winget install --id MongoDB.MongoDBServer -e

# Or download MSI from mongodb.com and include mongosh

macOS

brew tap mongodb/brew
brew install mongodb-community@7.0
brew services start mongodb-community@7.0

Linux

Ubuntu (example for 22.04)

curl -fsSL https://pgp.mongodb.com/server-7.0.asc | \
  sudo gpg -o /usr/share/keyrings/mongodb-server-7.0.gpg --dearmor

echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-7.0.gpg ] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/7.0 multiverse" | \
  sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list

sudo apt update
sudo apt install -y mongodb-org
sudo systemctl enable --now mongod

Docker

docker run -d --name mongo -p 27017:27017 -e MONGO_INITDB_ROOT_USERNAME=root -e MONGO_INITDB_ROOT_PASSWORD=secret mongo:7

Verify

mongosh
use admin
// create user
 db.createUser({ user: "app", pwd: "StrongP@ssw0rd", roles: ["readWriteAnyDatabase"] })
use app
 db.collection("t").insertOne({ name: "Ada" })
 db.collection("t").find()
