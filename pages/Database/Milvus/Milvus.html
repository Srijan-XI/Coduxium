Milvus

Milvus is an open-source vector database for high-scale similarity search over embeddings.

Overview

Vector DB AI/ML Similarity Search ANN Indexing

Milvus supports ANN indexing and retrieval workflows used in semantic search and retrieval-augmented generation systems.

Key Features

  • High-dimensional vector indexing and nearest-neighbor retrieval
  • Support for multiple index types and similarity metrics
  • Horizontal scalability for large embedding datasets
  • Ecosystem integrations with popular AI tooling

Getting Started

# Standalone quick start
docker run -d --name milvus-standalone -p 19530:19530 -p 9091:9091 milvusdb/milvus:v2.4.0

# Health check
docker logs milvus-standalone --tail 50

Use Cases

  • Semantic search and question-answer retrieval systems
  • Recommendation engines with embedding similarity
  • Multimodal retrieval over text and image vectors
