Memcached

Memcached is a high-performance distributed in-memory key-value cache designed to reduce database load and improve response latency.

Overview

Caching In-Memory Distributed KV Store

Memcached is best used as a simple ephemeral cache layer for computed results, hot query responses, and session-adjacent workloads.

Key Features

  • Very low-latency key-value operations
  • Simple protocol and broad client support
  • Horizontal scaling by sharding keys
  • Ideal for read-heavy traffic acceleration

Getting Started

# Docker quick start
docker run -d --name memcached -p 11211:11211 memcached:latest

# Health check
docker exec memcached sh -c "echo stats | nc 127.0.0.1 11211"
