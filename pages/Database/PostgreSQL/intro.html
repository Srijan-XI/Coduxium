What is PostgreSQL?

PostgreSQL (Postgres) is a highly reliable, standards-compliant RDBMS known for extensibility, strong concurrency control via MVCC, and rich data types.

Key Features

  • MVCC for consistent reads without read locks.
  • Advanced indexing: B-tree, Hash, GiST, GIN, BRIN.
  • Extensions: PostGIS, pg_trgm, hstore.
  • Rich types: JSONB, arrays, ranges, enums.
  • CTEs, window functions, and powerful query planner.

SQL Examples

Create role & database

CREATE ROLE app LOGIN PASSWORD 'StrongP@ssw0rd';
CREATE DATABASE appdb OWNER app;
GRANT ALL PRIVILEGES ON DATABASE appdb TO app;

Create table with index

\c appdb
CREATE TABLE users (
	id BIGSERIAL PRIMARY KEY,
	email TEXT NOT NULL UNIQUE,
	name TEXT NOT NULL,
	created_at TIMESTAMPTZ DEFAULT now()
);
CREATE INDEX idx_users_created_at ON users(created_at);

Query with CTE & window function

WITH ranked AS (
	SELECT id, name, created_at,
				 ROW_NUMBER() OVER (ORDER BY created_at DESC) AS rn
	FROM users
)
SELECT * FROM ranked WHERE rn <= 10;

JSONB Capabilities

CREATE TABLE events (
	id BIGSERIAL PRIMARY KEY,
	payload JSONB NOT NULL
);
CREATE INDEX idx_events_payload ON events USING GIN (payload);

-- Insert and query
INSERT INTO events(payload) VALUES ('{"type":"click","user":42}'::jsonb);
SELECT * FROM events WHERE payload @> '{"type":"click"}';
