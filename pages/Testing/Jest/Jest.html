Overview

Jest is a batteries-included test runner and assertion library for JavaScript and TypeScript. It runs tests in parallel, isolates modules, supports coverage, and integrates well with React, Node.js, and modern toolchains.

  • Zero-config for most JS apps
  • Fast parallel execution and watch mode
  • Powerful mocking and spies
  • Snapshot testing for UI/serializable output
  • TypeScript via Babel or ts-jest

Test Anatomy

// sum.js
export const sum = (a, b) => a + b

// sum.test.js
describe('sum', () => {
  test('adds two numbers', () => {
    expect(sum(2, 3)).toBe(5)
  })
})

Common Matchers

expect(value).toBe(3)
expect(value).toEqual({a:1})
expect(value).toBeTruthy()
expect(array).toContain('x')
await expect(promise).resolves.toBe('ok')

Mocking & Spies

const api = { fetch: () => 'real' }

test('mocks implementation', () => {
  const spy = jest.spyOn(api, 'fetch').mockReturnValue('fake')
  expect(api.fetch()).toBe('fake')
  spy.mockRestore()
})

Async Tests

test('awaits promise', async () => {
  const data = await Promise.resolve(42)
  expect(data).toBe(42)
})

Snapshot Testing

test('object snapshot', () => {
  const user = { id: 1, name: 'Ada' }
  expect(user).toMatchSnapshot()
})

Jest — Installation Guide

Set up Jest for JavaScript and TypeScript projects. Configure scripts, TypeScript, Babel, coverage, and debug.

NODE TS BABEL COVERAGE
🏠 Home Setup Run TypeScript Babel Coverage Debug

Setup

# Initialize project
npm init -y

# Install Jest	npm i -D jest

# Or yarn/pnpm
yarn add -D jest
pnpm add -D jest

# Optional: create config
npx jest --init

Package Scripts

{
  "scripts": {
    "test": "jest",
    "test:watch": "jest --watch",
    "test:ci": "jest --ci"
  }
}

Run Tests

# Run all tests
npm test

# Watch changed tests
npm run test:watch

# CI mode + JUnit report (example)
jest --ci --reporters=default --reporters=jest-junit

TypeScript

# Option A: ts-jest (no Babel)
npm i -D ts-jest @types/jest typescript
npx ts-jest config:init

# jest.config.ts
export default {
  testEnvironment: 'node',
  preset: 'ts-jest',
}
# Option B: Babel + TS
npm i -D @babel/core @babel/preset-env @babel/preset-typescript babel-jest @types/jest typescript

# babel.config.json
{
  "presets": ["@babel/preset-env", "@babel/preset-typescript"]
}

Babel (JS/TS)

// jest.config.js
module.exports = {
  transform: { '^.+\\.[tj]sx?$': ['babel-jest', { rootMode: 'upward' }] },
  testEnvironment: 'node',
}

Coverage

// jest.config.js
module.exports = {
  collectCoverage: true,
  collectCoverageFrom: ['src/**/*.{js,ts}'],
  coverageReporters: ['text', 'lcov'],
}
