📄 Overview

TOML (Tom's Obvious Minimal Language) is a minimal configuration file format designed to be easy to read and write. It's widely used in Rust projects (Cargo.toml) and Python packaging (pyproject.toml).

Key Features

  • Human-Readable — Clear and obvious semantics
  • Type System — Strong typing with clear data types
  • Minimal — Simple syntax, easy to learn
  • Unambiguous — Maps directly to hash tables

📖 Syntax Guide

Key-Value Pairs

# Basic values
title = "TOML Example"
name = "Tom Preston-Werner"
age = 43

# Booleans
enabled = true
disabled = false

# Numbers
integer = 42
float = 3.14
hex = 0xDEADBEEF
octal = 0o755
binary = 0b11010110

Strings

# Basic string
str = "Hello World"

# Multi-line string
multi = """
This is a
multi-line
string."""

# Literal string (no escape)
path = 'C:\Users\nodejs\templates'

# Multi-line literal
regex = '''
I [dw]on't need \d{2} apples'''

Arrays

# Arrays
integers = [1, 2, 3]
colors = ["red", "yellow", "green"]

# Multi-line arrays
data = [
  "alpha",
  "beta",
  "gamma"
]

# Nested arrays
matrix = [[1, 2], [3, 4]]

Tables (Objects)

# Define table
[database]
server = "192.168.1.1"
port = 5432
enabled = true

# Nested table
[database.connection]
max = 5000
timeout = 30

# Array of tables
[[products]]
name = "Hammer"
sku = 738594937

[[products]]
name = "Nail"
sku = 284758393

Inline Tables

name = { first = "Tom", last = "Preston-Werner" }
point = { x = 1, y = 2 }
animal = { type.name = "pug" }

Dates and Times

# Date-time
datetime = 1979-05-27T07:32:00Z
localtime = 07:32:00
localdate = 1979-05-27
localdatetime = 1979-05-27T07:32:00

💼 Real-World Examples

Cargo.toml (Rust)

[package]
name = "my-app"
version = "0.1.0"
edition = "2021"
authors = ["John Doe <john@example.com>"]

[dependencies]
tokio = { version = "1.0", features = ["full"] }
serde = { version = "1.0", features = ["derive"] }
actix-web = "4.0"

[dev-dependencies]
cargo-watch = "8.0"

[[bin]]
name = "server"
path = "src/server.rs"

pyproject.toml (Python)

[build-system]
requires = ["setuptools>=61.0"]
build-backend = "setuptools.build_meta"

[project]
name = "myapp"
version = "1.0.0"
description = "My application"
authors = [
    {name = "John Doe", email = "john@example.com"}
]
dependencies = [
    "flask>=2.0",
    "requests>=2.28"
]

[project.optional-dependencies]
dev = ["pytest", "black", "mypy"]

[tool.black]
line-length = 88
target-version = ['py311']

[tool.pytest.ini_options]
testpaths = ["tests"]

Application Config

[app]
name = "MyApp"
version = "1.0.0"
debug = false

[server]
host = "0.0.0.0"
port = 8080
workers = 4

[database]
url = "postgresql://localhost/mydb"
pool_size = 10

[logging]
level = "info"
format = "json"
file = "/var/log/app.log"

Using TOML in Code

# Python - pip install toml
import toml

# Parse TOML
with open('config.toml', 'r') as f:
    config = toml.load(f)

# Write TOML
data = {'title': 'Example', 'value': 42}
with open('output.toml', 'w') as f:
    toml.dump(data, f)

# Rust
use toml;

let config: Config = toml::from_str(&toml_string)?;

# JavaScript - npm install @iarna/toml
const toml = require('@iarna/toml');
const fs = require('fs');

const config = toml.parse(fs.readFileSync('config.toml', 'utf8'));
const tomlString = toml.stringify(config);
