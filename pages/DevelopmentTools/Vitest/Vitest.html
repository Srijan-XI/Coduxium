🧪 Overview

Vitest is a blazing fast unit test framework powered by Vite. It provides instant feedback with watch mode, native ESM support, and Jest-compatible API.

Key Features

  • Lightning Fast — Powered by Vite, instant test execution
  • Jest Compatible — Familiar API, easy migration
  • Native ESM — First-class ES modules support
  • Smart Watch Mode — Instant rerun on file changes
  • TypeScript — Out-of-the-box TypeScript support
  • UI Mode — Beautiful test runner UI
  • Component Testing — Test React, Vue, Svelte components

📦 Installation

# Install Vitest
npm install -D vitest

# With UI
npm install -D vitest @vitest/ui

# Add to package.json
{
  "scripts": {
    "test": "vitest",
    "test:ui": "vitest --ui",
    "test:coverage": "vitest --coverage"
  }
}

Configuration

// vite.config.ts
import { defineConfig } from 'vitest/config'

export default defineConfig({
  test: {
    globals: true,
    environment: 'jsdom',
    setupFiles: './src/test/setup.ts',
    coverage: {
      provider: 'v8',
      reporter: ['text', 'html', 'json']
    }
  }
})

📖 Basic Usage

Writing Tests

// math.test.ts
import { describe, it, expect } from 'vitest'

describe('Math operations', () => {
  it('adds two numbers', () => {
    expect(1 + 1).toBe(2)
  })

  it('multiplies numbers', () => {
    expect(2 * 3).toBe(6)
  })
})

Async Tests

import { describe, it, expect } from 'vitest'

describe('Async operations', () => {
  it('fetches data', async () => {
    const data = await fetchData()
    expect(data).toBeDefined()
  })

  it('resolves promise', () => {
    return expect(Promise.resolve(42)).resolves.toBe(42)
  })
})

Matchers

// Equality
expect(value).toBe(expected)
expect(value).toEqual(expected)

// Truthiness
expect(value).toBeTruthy()
expect(value).toBeFalsy()

// Numbers
expect(value).toBeGreaterThan(3)
expect(value).toBeLessThan(5)

// Strings
expect(string).toContain('text')
expect(string).toMatch(/pattern/)

// Arrays
expect(array).toContain(item)
expect(array).toHaveLength(3)

// Objects
expect(obj).toHaveProperty('key')
expect(obj).toMatchObject({ key: 'value' })

Mocking

import { vi } from 'vitest'

// Mock functions
const mockFn = vi.fn()
mockFn('hello')
expect(mockFn).toHaveBeenCalledWith('hello')

// Mock modules
vi.mock('./api', () => ({
  fetchUser: vi.fn(() => ({ name: 'John' }))
}))

// Spy on methods
const spy = vi.spyOn(obj, 'method')
spy.mockReturnValue(42)

Snapshot Testing

import { expect, test } from 'vitest'

test('renders correctly', () => {
  const result = render()
  expect(result).toMatchSnapshot()
})

// Update snapshots
npm run test -- -u

🚀 Advanced Features

Component Testing (React)

import { render, screen } from '@testing-library/react'
import { expect, test } from 'vitest'
import Button from './Button'

test('renders button', () => {
  render(<Button>Click me</Button>)
  expect(screen.getByText('Click me')).toBeInTheDocument()
})

Coverage

# Install coverage provider
npm install -D @vitest/coverage-v8

# Run with coverage
npm run test:coverage

# Configure in vite.config.ts
export default defineConfig({
  test: {
    coverage: {
      provider: 'v8',
      reporter: ['text', 'html', 'lcov'],
      exclude: ['**/node_modules/**', '**/dist/**']
    }
  }
})

Watch Mode

# Run in watch mode (default)
npm run test

# Watch specific files
vitest src/components

# Run once
vitest run

UI Mode

# Start UI
npm run test:ui

# Access at http://localhost:51204/__vitest__/
