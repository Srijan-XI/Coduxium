Introduction

What is Zig?

Zig is a general-purpose systems programming language designed around explicitness and reliability. It avoids hidden control flow, hidden allocations, and hidden preprocessor behavior while still delivering low-level control.

Key Features

Explicit Errors - no hidden exceptions
comptime - compile-time metaprogramming
C Interop - easy integration with existing libraries
Cross Compile - targets from one toolchain
Single Binary Tooling - build, test, format, docs
Allocator APIs - deliberate memory strategy

Where Zig Fits Best

  • CLI tools and native developer tooling
  • Latency-sensitive network services
  • Embedded and cross-platform runtime components
  • Incremental migration of C/C++ codebases

Installation & Setup

Install Zig

# Windows
winget install zig.zig

# macOS
brew install zig

# Linux (example)
sudo snap install zig --classic

Verify Installation

zig version
zig env

Project Bootstrap

mkdir zig-demo
cd zig-demo
zig init
zig build run
zig test src/root.zig

User Guide

Hello World

const std = @import("std");

pub fn main() !void {
    const out = std.io.getStdOut().writer();
    try out.print("Hello from Zig!\n", .{});
}

Error Handling Pattern

fn divide(a: i32, b: i32) !i32 {
    if (b == 0) return error.DivideByZero;
    return @divTrunc(a, b);
}

pub fn main() !void {
    const result = try divide(12, 3);
    _ = result;
}

Allocator-based Dynamic Memory

const std = @import("std");

pub fn main() !void {
    var gpa = std.heap.GeneralPurposeAllocator(.{}){};
    defer _ = gpa.deinit();
    const allocator = gpa.allocator();

    var list = std.ArrayList(u8).init(allocator);
    defer list.deinit();

    try list.appendSlice("zig");
}
