📚 Introduction

What is MATLAB?

MATLAB (Matrix Laboratory) is a proprietary multi-paradigm programming language and numeric computing environment developed by MathWorks. First released in the late 1970s, MATLAB has become the standard tool for engineers and scientists for algorithm development, data analysis, visualization, and numerical computation.

Key Features

📊 Matrix Operations — Built for linear algebra
📈 Visualization — Advanced plotting and graphics
🧮 Numerical Computing — High-performance calculations
🔧 Toolboxes — Specialized for engineering domains
🤖 Simulink — Visual programming for simulations
📡 Signal Processing — Audio, image, and data analysis
🎯 Algorithm Development — Rapid prototyping
🔄 Integration — C/C++, Python, Java interop

Why Choose MATLAB?

  • Industry Standard: Used in aerospace, automotive, finance, biotech
  • Comprehensive Toolboxes: Control systems, signal processing, AI, more
  • Easy Visualization: Publication-quality plots with minimal code
  • Rapid Prototyping: Test algorithms quickly
  • Educational: Taught in universities worldwide
  • Integration: Works with hardware, embedded systems
⚠️ Note: MATLAB is commercial software. Consider open alternatives like GNU Octave for similar functionality.

⚙️ Installation & Setup

Get started with MATLAB by installing it from MathWorks.

Windows Installation

1
Create MathWorks Account

Visit mathworks.com and create an account. You'll need a valid license (academic, trial, or commercial).

2
Download Installer

Log in and download the MATLAB installer for Windows from your account.

3
Run Installer

Execute the installer and sign in with your MathWorks credentials. Select the toolboxes you need during installation.

4
Activate License

Follow the activation prompts to activate your MATLAB license.

Academic Access

Many universities provide MATLAB access to students. Check with your institution's IT department or campus license portal.

macOS Installation

Download the macOS installer from MathWorks Downloads and follow the installation wizard. Ensure your Mac meets the system requirements.

System Requirements

  • macOS 11 (Big Sur) or later
  • 8 GB RAM minimum (16 GB recommended)
  • 3-5 GB disk space for base installation

Linux Installation

1
Download Installer

Download the Linux installer from MathWorks website.

2
Extract and Run 
unzip matlab_installer.zip
cd matlab_installer
./install
3
Follow GUI Installer

The graphical installer will guide you through the setup process.

Alternative: GNU Octave (Open Source)

# Ubuntu/Debian
sudo apt install octave

# Fedora
sudo dnf install octave

# Arch Linux
sudo pacman -S octave

First Launch

After installation, launch MATLAB. You'll see:

  • Command Window: Execute commands interactively
  • Workspace: View variables in memory
  • Current Folder: Browse files
  • Editor: Write and save scripts (.m files)

📖 User Guide

Hello World

% Display message
disp('Hello, Coduxium XI!')

% Alternative
fprintf('Welcome to MATLAB!\n')

Variables & Basic Operations

% Variables (no declaration needed)
x = 5;
y = 10;
z = x + y;

% Display variable
disp(z)

% Suppress output with semicolon
a = 100;  % No output

% Multiple assignments
p = 1; q = 2; r = 3;

Matrices & Arrays

% Create matrix
A = [1 2 3; 4 5 6; 7 8 9];

% Row vector
v = [1 2 3 4 5];

% Column vector
c = [1; 2; 3; 4; 5];

% Matrix operations
B = A';              % Transpose
C = A * A;           % Matrix multiplication
D = A .* A;          % Element-wise multiplication

% Access elements
element = A(2, 3);   % Row 2, Column 3
row = A(1, :);       % First row
col = A(:, 2);       % Second column

Common Functions

% Generate sequences
seq = 1:10;              % 1 to 10
seq2 = 0:0.1:1;          % 0 to 1, step 0.1
lin = linspace(0, 10, 50); % 50 points from 0 to 10

% Matrix creation
zeros_mat = zeros(3, 4);  % 3x4 matrix of zeros
ones_mat = ones(2, 5);    % 2x5 matrix of ones
eye_mat = eye(4);         % 4x4 identity matrix
rand_mat = rand(3, 3);    % 3x3 random matrix

% Math functions
sqrt_val = sqrt(16);
sine = sin(pi/2);
log_val = log(10);
abs_val = abs(-5);

Control Flow

% If-else
x = 10;
if x > 5
    disp('x is greater than 5')
elseif x == 5
    disp('x equals 5')
else
    disp('x is less than 5')
end

% For loop
for i = 1:5
    fprintf('Iteration %d\n', i)
end

% While loop
count = 1;
while count <= 5
    disp(count)
    count = count + 1;
end

Plotting & Visualization

% Simple plot
x = 0:0.1:10;
y = sin(x);
plot(x, y)
title('Sine Wave')
xlabel('X-axis')
ylabel('Y-axis')
grid on

% Multiple plots
figure;
subplot(2, 1, 1)
plot(x, sin(x))
title('Sine')

subplot(2, 1, 2)
plot(x, cos(x))
title('Cosine')

% 3D plot
[X, Y] = meshgrid(-5:0.5:5);
Z = sin(sqrt(X.^2 + Y.^2));
surf(X, Y, Z)
title('3D Surface')

Functions (M-files)

% Save as myFunction.m
function result = myFunction(a, b)
    % Calculate sum and product
    result = a + b * 2;
end

% Usage
output = myFunction(5, 10);

% Multiple outputs
function [sum_val, prod_val] = calculate(x, y)
    sum_val = x + y;
    prod_val = x * y;
end

[s, p] = calculate(4, 5);

Working with Data

% Read CSV
data = readmatrix('data.csv');

% Read Excel
xlsdata = readtable('data.xlsx');

% Save workspace
save('mydata.mat', 'A', 'B');

% Load workspace
load('mydata.mat');

% Export data
writematrix(A, 'output.csv');

Symbolic Math

% Requires Symbolic Math Toolbox
syms x y
f = x^2 + 2*x + 1;

% Solve equation
solution = solve(f == 0, x);

% Differentiate
df = diff(f, x);

% Integrate
int_f = int(f, x);

GPU Computing

% Transfer array to GPU
A_gpu = gpuArray(rand(1000));
B_gpu = gpuArray(rand(1000));

% Perform computation on GPU
C_gpu = A_gpu * B_gpu;

% Transfer back to CPU
C = gather(C_gpu);

Machine Learning

% Load sample data
load fisheriris

% Train SVM classifier
SVMModel = fitcsvm(meas, species);

% Make predictions
predictions = predict(SVMModel, meas(1:10, :));
