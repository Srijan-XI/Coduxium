📚 Introduction

What is Scala?

Scala (Scalable Language) is a modern, statically-typed programming language designed by Martin Odersky and first released in 2004. It seamlessly integrates object-oriented and functional programming paradigms, running on the Java Virtual Machine (JVM) and offering full interoperability with Java. Scala is designed to be concise, elegant, and type-safe.

Key Features

🎯 Multi-Paradigm — OOP + Functional Programming
☕ JVM Compatible — Use Java libraries seamlessly
🔒 Type-Safe — Advanced type system
⚡ Concise Syntax — Less boilerplate than Java
🔄 Immutability — Functional programming focus
🧵 Concurrency — Akka actors, futures
📊 Big Data — Apache Spark foundation
🔧 Pattern Matching — Powerful control flow

Why Choose Scala?

  • Functional Programming: First-class functions, immutability, monads
  • Java Ecosystem: Access to millions of Java libraries
  • Concurrency: Built-in support for concurrent programming
  • Big Data: Apache Spark, the leading big data framework
  • Type Safety: Catch errors at compile time
  • Career Opportunities: High-paying jobs in fintech, big data
⚠️ Note: Scala 3 (Dotty) introduced significant improvements. New projects should use Scala 3.

Functions

// Basic function
def greet(name: String): String = {
  s"Hello, $name!"
}

// Single expression (no braces needed)
def add(a: Int, b: Int): Int = a + b

// Default parameters
def greetWithTitle(name: String, title: String = "Mr."): String =
  s"Hello, $title $name!"

// Anonymous functions (lambdas)
val square = (x: Int) => x * x
val double: Int => Int = _ * 2

// Call functions
println(greet("Alice"))
println(add(5, 10))
println(square(7))

Collections

Pattern Matching

// Basic pattern matching
def describe(x: Any): String = x match {
  case 0 => "zero"
  case 1 => "one"
  case s: String => s"String: $s"
  case i: Int if i > 1 => s"Integer greater than 1: $i"
  case _ => "something else"
}

// Pattern matching with case classes
case class Person(name: String, age: Int)

def greetPerson(p: Person): String = p match {
  case Person(name, age) if age < 18 => s"Hi $name, you're a minor"
  case Person(name, age) => s"Hello $name, you're $age years old"
}

// List patterns
def sumList(list: List[Int]): Int = list match {
  case Nil => 0
  case head :: tail => head + sumList(tail)
}

Classes & Objects

// Class
class Person(val name: String, var age: Int) {
  def greet(): String = s"Hi, I'm $name"
  
  def birthday(): Unit = {
    age += 1
    println(s"Happy birthday! Now $age years old")
  }
}

// Create instance
val person = Person("Alice", 25)
person.greet()
person.birthday()

// Case class (immutable data)
case class Point(x: Int, y: Int)
val p1 = Point(1, 2)
val p2 = p1.copy(x = 3) // New instance

// Singleton object
object MathUtils {
  def square(x: Int): Int = x * x
  val PI = 3.14159
}

println(MathUtils.square(5))

Higher-Order Functions

// Functions as parameters
def applyTwice(f: Int => Int, x: Int): Int = f(f(x))
val increment = (x: Int) => x + 1
println(applyTwice(increment, 5)) // 7

// Map, filter, reduce
val numbers = List(1, 2, 3, 4, 5)
val squared = numbers.map(x => x * x)
val evens = numbers.filter(_ % 2 == 0)
val sum = numbers.reduce(_ + _)

// For comprehensions
val result = for {
  x <- 1 to 3
  y <- 1 to 2
} yield (x, y)
// List((1,1), (1,2), (2,1), (2,2), (3,1), (3,2))

Options & Error Handling

// Option type (like Rust's Option or Java's Optional)
def findPerson(id: Int): Option[Person] = {
  if (id > 0) Some(Person("Alice", 25))
  else None
}

val person = findPerson(1)
person match {
  case Some(p) => println(p.name)
  case None => println("Not found")
}

// Try for exception handling
import scala.util.{Try, Success, Failure}

def divide(a: Int, b: Int): Try[Int] = Try(a / b)

divide(10, 2) match {
  case Success(result) => println(s"Result: $result")
  case Failure(e) => println(s"Error: ${e.getMessage}")
}

Futures & Async Programming

import scala.concurrent.{Future, Await}
import scala.concurrent.ExecutionContext.Implicits.global
import scala.concurrent.duration._

// Create future
val future: Future[Int] = Future {
  Thread.sleep(1000)
  42
}

// Transform future
val doubled = future.map(_ * 2)

// Combine futures
val f1 = Future { 10 }
val f2 = Future { 20 }
val combined = for {
  a <- f1
  b <- f2
} yield a + b

// Wait for result (blocking)
val result = Await.result(combined, 5.seconds)
println(result) // 30

Implicits (Scala 2) / Given/Using (Scala 3)

// Scala 3 style
trait Ordering[T]:
  def compare(a: T, b: T): Int

given intOrdering: Ordering[Int] with
  def compare(a: Int, b: Int): Int = a - b

def max[T](a: T, b: T)(using ord: Ordering[T]): T =
  if ord.compare(a, b) > 0 then a else b

println(max(5, 10)) // 10 (uses intOrdering)

Type Classes

// Define type class
trait Show[T]:
  def show(value: T): String

// Implement for Int
given Show[Int] with
  def show(value: Int): String = s"Int($value)"

// Implement for String
given Show[String] with
  def show(value: String): String = s"String($value)"

// Generic function using type class
def display[T](value: T)(using s: Show[T]): Unit =
  println(s.show(value))

display(42)
display("hello")
