📚 Introduction

What is Java?

Java is a class-based, object-oriented programming language designed for portability and reliability. Created by Sun Microsystems (now Oracle) in 1995, Java powers billions of devices worldwide – from Android apps to enterprise backends, big data platforms to microservices.

Key Features

☕ Object-Oriented — Everything is an object
🌐 Platform Independent — Write once, run anywhere
🗑️ Garbage Collection — Automatic memory management
🔒 Strongly Typed — Type safety at compile time
📚 Rich Standard Library — Comprehensive built-in APIs
🧵 Multithreading — Native concurrency support
🔐 Security — Built-in security features
⚡ Performance — JIT compilation optimization

Why Choose Java?

  • Portability: JVM enables cross-platform development
  • Enterprise Ready: Powers major financial and e-commerce systems
  • Android Development: Primary language for Android apps
  • Rich Ecosystem: Maven, Gradle, Spring, Hibernate
  • Job Market: High demand in enterprise sector
  • Mature & Stable: 25+ years of proven reliability
⚠️ Note: Java requires JDK (Java Development Kit) for development or JRE (Java Runtime Environment) for running Java applications.

macOS Installation

Using Homebrew

# Install OpenJDK
brew install openjdk@21

# Symlink for system Java
sudo ln -sfn $(brew --prefix)/opt/openjdk@21/libexec/openjdk.jdk \
  /Library/Java/JavaVirtualMachines/openjdk-21.jdk

# Set JAVA_HOME (add to ~/.zshrc)
export JAVA_HOME=$(/usr/libexec/java_home -v 21)
export PATH="$JAVA_HOME/bin:$PATH"

# Reload shell
source ~/.zshrc

# Verify
java --version

Classes & Objects

public class Person {
    // Fields
    private String name;
    private int age;
    
    // Constructor
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // Methods
    public void introduce() {
        System.out.println("Hi, I'm " + name);
    }
    
    // Getters & Setters
    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
}

// Usage
Person person = new Person("Alice", 30);
person.introduce();

Collections

import java.util.*;

languages.add("Python");
languages.add("Go");

Map scores = new HashMap<>();
scores.put("Alice", 95);

    System.out.println(lang);

    .forEach(System.out::println);

Inheritance & Polymorphism

// Parent class
public class Animal {
    public void makeSound() {
        System.out.println("Some sound");
    }
}

// Child class
public class Dog extends Animal {
    @Override
    public void makeSound() {
        System.out.println("Woof!");
    }
    
    public void fetch() {
        System.out.println("Fetching...");
    }
}

// Usage
Animal myDog = new Dog();
myDog.makeSound();  // Woof!

Interfaces

// Interface
public interface Drawable {
    void draw();  // Abstract method
    
    default void describe() {  // Default method (Java 8+)
        System.out.println("This is drawable");
    }
}

// Implementation
public class Circle implements Drawable {
    @Override
    public void draw() {
        System.out.println("Drawing circle");
    }
}

Circle circle = new Circle();
circle.draw();

Exception Handling

try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.err.println("Cannot divide by zero!");
} finally {
    System.out.println("Cleanup");
}

// Multi-catch (Java 7+)
try {
    // Code
} catch (IOException | SQLException e) {
    e.printStackTrace();
}

// Try-with-resources (Java 7+)
try (BufferedReader br = new BufferedReader(new FileReader("file.txt"))) {
    String line = br.readLine();
} catch (IOException e) {
    e.printStackTrace();
}

Lambda Expressions (Java 8+)

// Before Java 8
List names = Arrays.asList("Alice", "Bob", "Charlie");
names.forEach(new Consumer() {
    public void accept(String name) {
        System.out.println(name);
    }
});

// With Lambda
names.forEach(name -> System.out.println(name));

// Method reference
names.forEach(System.out::println);

// Functional interface
Runnable task = () -> System.out.println("Running...");
new Thread(task).start();

Stream API

List numbers = Arrays.asList(1, 2, 3, 4, 5);

// Filter and sum
int sum = numbers.stream()
    .filter(n -> n % 2 == 0)
    .mapToInt(Integer::intValue)
    .sum();

System.out.println("Sum of evens: " + sum);
