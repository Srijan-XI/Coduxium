📚 Introduction

What is Dart & Flutter?

Dart is an open-source, client-optimized programming language developed by Google. It's designed for building fast apps on any platform. Flutter is Google's UI toolkit built with Dart that allows developers to create beautiful, natively compiled applications for mobile, web, and desktop from a single codebase.

Key Features

📱 Cross-Platform — iOS, Android, Web, Desktop
⚡ Hot Reload — See changes instantly
🎨 Beautiful UI — Rich widget catalog
🚀 High Performance — Native ARM code compilation
🧩 Widget-Based — Everything is a widget
📦 Package Ecosystem — pub.dev repository
🔧 Developer Tools — DevTools, Dart analyzer
🌐 Google Backed — Strong community support

Why Choose Dart & Flutter?

  • Single Codebase: Write once, deploy everywhere
  • Fast Development: Hot reload speeds up iteration
  • Native Performance: Compiled to native ARM code
  • Expressive UI: Create custom, beautiful designs easily
  • Growing Ecosystem: 30,000+ packages on pub.dev
  • Career Opportunities: High demand for Flutter developers
⚠️ Important: Flutter 3.0+ supports all platforms. Always use the latest stable version for new projects.

⚙️ Installation & Setup

Get started with Flutter by installing the SDK on your system.

Windows Installation

1
Download Flutter SDK

Visit Flutter Windows Install and download the latest stable release.

2
Extract SDK

Extract the zip file to a location like C:\flutter (avoid Program Files).

3
Add to PATH

Add C:\flutter\bin to your system PATH environment variable.

4
Run Flutter Doctor 
flutter doctor

This command checks your environment and displays a report of the status.

5
Install Dependencies
  • Android Studio: For Android development
  • Visual Studio: For Windows desktop development
  • Chrome: For web development

macOS Installation

Method 1: Direct Download

1
Download Flutter SDK

Download from Flutter macOS Install

2
Extract and Add to PATH 
cd ~/development
unzip ~/Downloads/flutter_macos_*.zip

# Add to PATH in ~/.zshrc or ~/.bash_profile
export PATH="$PATH:`pwd`/flutter/bin"

source ~/.zshrc
3
Run Flutter Doctor 
flutter doctor

Install Development Tools

  • Xcode: For iOS/macOS development
  • CocoaPods: sudo gem install cocoapods
  • Android Studio: For Android development

Linux Installation

Method 1: Snap (Recommended)

sudo snap install flutter --classic

# Verify installation
flutter doctor

Method 2: Manual Installation

cd ~/development
wget https://storage.googleapis.com/flutter_infra_release/releases/stable/linux/flutter_linux_*-stable.tar.xz
tar xf flutter_linux_*-stable.tar.xz

# Add to PATH
export PATH="$PATH:~/development/flutter/bin"

flutter doctor

Install Dependencies

# Ubuntu/Debian
sudo apt install clang cmake ninja-build libgtk-3-dev

# Install Android Studio for Android development

IDE Setup

Visual Studio Code (Recommended)

# Install Flutter and Dart extensions
# Open VS Code > Extensions > Search "Flutter"
# Install both Flutter and Dart extensions

Android Studio

Install Flutter and Dart plugins from Settings → Plugins.

Create First App

# Create new Flutter app
flutter create my_first_app
cd my_first_app

# Run on connected device/emulator
flutter run

Stateful Widget

class Counter extends StatefulWidget {
  @override
  _CounterState createState() => _CounterState();
}

class _CounterState extends State {
  int _count = 0;

  void _increment() {
    setState(() {
      _count++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      mainAxisAlignment: MainAxisAlignment.center,
      children: [
        Text('Count: $_count', style: TextStyle(fontSize: 30)),
        SizedBox(height: 20),
        ElevatedButton(
          onPressed: _increment,
          child: Text('Increment'),
        ),
      ],
    );
  }
}

State Management with Provider

import 'package:provider/provider.dart';

class CounterModel extends ChangeNotifier {
  int _count = 0;
  int get count => _count;

  void increment() {
    _count++;
    notifyListeners();
  }
}

// Provide model
ChangeNotifierProvider(
  create: (context) => CounterModel(),
  child: MyApp(),
)

// Consume model
Consumer(
  builder: (context, counter, child) {
    return Text('${counter.count}');
  },
)

Async Programming

// Future
Future fetchData() async {
  await Future.delayed(Duration(seconds: 2));
  return 'Data loaded';
}

// FutureBuilder
FutureBuilder(
  future: fetchData(),
  builder: (context, snapshot) {
    if (snapshot.connectionState == ConnectionState.waiting) {
      return CircularProgressIndicator();
    }
    if (snapshot.hasError) {
      return Text('Error: ${snapshot.error}');
    }
    return Text('Data: ${snapshot.data}');
  },
)
