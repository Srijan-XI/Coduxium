📚 Introduction

What is JavaScript?

JavaScript is a high-level, interpreted programming language that conforms to the ECMAScript specification. Created in 1995 by Brendan Eich, it powers the interactive web and has evolved into a full-stack language capable of running on servers, mobile devices, and beyond.

Key Features

🌐 Browser Native — Runs in all web browsers
⚡ Event-Driven — Perfect for interactive UIs
🔄 Asynchronous — Promises, async/await
🎯 Dynamic Typing — Flexible variable types
📦 First-Class Functions — Functions as values
🔧 Prototype-Based — Object inheritance
🌍 Universal — Frontend & backend
📈 Growing Ecosystem — NPM with 2M+ packages

Why Choose JavaScript?

  • Ubiquitous: Required skill for web development
  • Full-Stack: One language for frontend and backend (Node.js)
  • Rich Ecosystem: React, Vue, Angular, Express, Next.js
  • Active Community: Largest developer community worldwide
  • Job Market: Highest demand programming language
  • Modern Features: ES6+ brings class syntax, modules, arrow functions
⚠️ Note: Modern JavaScript (ES6+) introduced significant improvements. Always use modern syntax and features for new projects.

📖 User Guide

Variables & Constants

// Modern variable declarations (ES6+)
let name = "Alice";           // Block-scoped, reassignable
const PI = 3.14159;           // Block-scoped, constant
var oldWay = "avoid";         // Function-scoped (legacy)

// Data types
let str = "Hello";            // String
let num = 42;                 // Number
let bool = true;              // Boolean
let arr = [1, 2, 3];          // Array
let obj = { key: "value" };   // Object
let nothing = null;           // Null
let undef = undefined;        // Undefined

Functions

// Function declaration
function greet(name) {
    return `Hello, ${name}!`;
}

// Arrow function (ES6+)
const greet = (name) => `Hello, ${name}!`;

// Arrow function with block
const greet = (name) => {
    console.log("Greeting...");
    return `Hello, ${name}!`;
};

// Default parameters
const greet = (name = "Guest") => `Hello, ${name}!`;

console.log(greet("Alice"));  // Hello, Alice!

Template Literals

const name = "Alice";
const age = 30;

// Template literals (ES6+)
const message = `My name is ${name} and I'm ${age} years old`;

// Multi-line strings
const html = `
  <div>
    <h1>${name}</h1>
    <p>Age: ${age}</p>
  </div>
`;

Arrays & Methods

const fruits = ["apple", "banana", "cherry"];

// Modern array methods
fruits.push("orange");              // Add to end
fruits.pop();                       // Remove from end
fruits.map(f => f.toUpperCase());   // Transform
fruits.filter(f => f.length > 5);   // Filter
fruits.find(f => f.startsWith("a"));// Find first
fruits.forEach(f => console.log(f));// Iterate

// Destructuring
const [first, second] = fruits;

// Spread operator
const moreFruits = [...fruits, "grape", "mango"];

Objects

// Object literal
const person = {
    name: "Alice",
    age: 30,
    greet() {
        return `Hi, I'm ${this.name}`;
    }
};

// Accessing properties
console.log(person.name);
console.log(person["age"]);

// Destructuring
const { name, age } = person;

// Spread operator
const updatedPerson = { ...person, city: "NYC" };

// Object methods
Object.keys(person);      // ["name", "age", "greet"]
Object.values(person);    // Values array
Object.entries(person);   // Key-value pairs

Control Flow

// If-else
const age = 18;
if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

// Ternary operator
const status = age >= 18 ? "Adult" : "Minor";

// For loops
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// For...of (arrays)
for (const fruit of fruits) {
    console.log(fruit);
}

// For...in (objects)
for (const key in person) {
    console.log(key, person[key]);
}

Promises & Async/Await

// Promise
fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));

// Async/Await (Modern approach)
async function fetchData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error(error);
    }
}

fetchData();

Classes (ES6+)

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    
    greet() {
        return `Hi, I'm ${this.name}`;
    }
    
    static species() {
        return "Homo sapiens";
    }
}

// Inheritance
class Student extends Person {
    constructor(name, age, studentId) {
        super(name, age);
        this.studentId = studentId;
    }
    
    study() {
        return `${this.name} is studying`;
    }
}

const student = new Student("Bob", 20, "S123");
console.log(student.greet());
console.log(student.study());

Modules (ES6)

// Export (utils.js)
export const add = (a, b) => a + b;
export const multiply = (a, b) => a * b;
export default class Calculator { }

// Import (main.js)
import Calculator, { add, multiply } from './utils.js';

console.log(add(2, 3));  // 5
