📚 Introduction

What is Rust?

Rust is a systems programming language focused on safety, speed, and concurrency. Created by Mozilla, Rust guarantees memory safety without a garbage collector, making it ideal for performance-critical applications, embedded systems, and WebAssembly.

Key Features

🛡️ Memory Safety — No null, no dangling pointers
⚡ Zero-Cost Abstractions — No runtime overhead
🔒 Ownership System — Compile-time memory management
🧵 Fearless Concurrency — Thread safety guaranteed
📦 Cargo — Built-in package manager
🌐 Cross-Platform — Windows, Linux, macOS, web

Why Choose Rust?

  • Safety: Eliminates entire classes of bugs
  • Performance: As fast as C/C++
  • Productivity: Great tooling and error messages
  • Growing: Most loved language (Stack Overflow survey)

⚙️ Installation & Setup

Install Rust (All Platforms)

# Install rustup (Rust toolchain installer)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Windows: Download from rustup.rs

# Verify
rustc --version
cargo --version

Your First Rust Program

    println!("Hello, Rust!");
}

// Or use Cargo
cargo new hello_world
cd hello_world

Ownership & Borrowing

fn main() {
    // Ownership
    let s1 = String::from("hello");
    let s2 = s1; // s1 moved to s2, s1 no longer valid
    
    // Borrowing (immutable reference)
    let s3 = String::from("world");
    let len = calculate_length(&s3);
    println!("{} has length {}", s3, len);
    
    // Mutable reference
    let mut s4 = String::from("hello");
    change(&mut s4);
}

fn calculate_length(s: &String) -> usize {
    s.len()
}

fn change(s: &mut String) {
    s.push_str(", world");
}

Structs & Enums

// Struct
struct Person {
    name: String,
    age: u32,
}

impl Person {
    fn new(name: String, age: u32) -> Person {
        Person { name, age }
    }
    
    fn greet(&self) {
        println!("Hi, I'm {}", self.name);
    }
}

// Enum
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
}

fn main() {
    let person = Person::new(String::from("Alice"), 30);
    person.greet();
}

Pattern Matching

fn main() {
    let number = 7;
    
    match number {
        1 => println!("One"),
        2 | 3 | 5 | 7 => println!("Prime"),
        _ => println!("Other"),
    }
    
    // Option handling
    let some_number = Some(5);
    match some_number {
        Some(x) => println!("Got: {}", x),
        None => println!("Nothing"),
    }
}
