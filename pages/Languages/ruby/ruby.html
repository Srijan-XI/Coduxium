📚 Introduction

What is Ruby?

Ruby is a dynamic, object-oriented programming language focused on simplicity and productivity. Created by Yukihiro Matsumoto in 1995, Ruby powers the popular Ruby on Rails web framework and emphasizes developer happiness.

Key Features

🎨 Expressive — Natural, readable syntax
🔧 Object-Oriented — Everything is an object
📦 Gems — RubyGems package manager
🚂 Rails — Powerful web framework
🔄 Dynamic — Flexible and malleable
🌐 Cross-Platform — Windows, macOS, Linux

Why Choose Ruby?

  • Ruby on Rails: Build web apps quickly
  • Readable: Code reads like English
  • Productive: Write less, accomplish more
  • Community: Friendly, helpful developers

⚙️ Installation & Setup

Windows

# Download RubyInstaller
# https://rubyinstaller.org/

# Or use winget
winget install RubyInstallerTeam.Ruby

# Verify
ruby --version

macOS

# Using Homebrew
brew install ruby

# Add to PATH (~/.zshrc)
export PATH="/opt/homebrew/opt/ruby/bin:$PATH"

# Verify
ruby --version

Linux

# Ubuntu/Debian
sudo apt update
sudo apt install ruby-full

# Fedora
sudo dnf install ruby

# Verify
ruby --version

Your First Ruby Program

# hello.rb
puts "Hello, Ruby!"

# Run
ruby hello.rb

📖 User Guide

Variables & Data Types

# Variables
name = "Alice"
age = 30
price = 99.99
active = true

# Symbols
:status
:name

# Arrays
fruits = ["apple", "banana", "cherry"]
fruits << "orange"  # Append

# Hashes
person = {
  name: "Alice",
  age: 30,
  email: "alice@example.com"
}

puts person[:name]

Methods (Functions)

# Method definition
def greet(name)
  "Hello, #{name}!"
end

puts greet("Alice")

# Default parameters
def power(base, exponent = 2)
  base ** exponent
end

# Blocks
[1, 2, 3].each do |num|
  puts num
end

# Short syntax
[1, 2, 3].each { |num| puts num }

Classes

class Person
  attr_accessor :name, :age
  
  def initialize(name, age)
    @name = name
    @age = age
  end
  
  def greet
    "Hi, I'm #{@name}"
  end
end

person = Person.new("Alice", 30)
puts person.greet
puts person.name

Enumerable Methods

numbers = [1, 2, 3, 4, 5]

# Map
squared = numbers.map { |n| n ** 2 }

# Select (filter)
evens = numbers.select { |n| n.even? }

# Reduce
sum = numbers.reduce(0) { |acc, n| acc + n }

# Chain methods
result = numbers
  .select { |n| n.even? }
  .map { |n| n ** 2 }
  .reduce(:+)

🚀 Advanced Topics

Ruby on Rails

# Install Rails
gem install rails

# Create new app
rails new myapp

# Start server
cd myapp
rails server

# Visit: http://localhost:3000

Gems (Package Manager)

# Install gem
gem install nokogiri

# List installed gems
gem list

# In Gemfile
gem 'rails', '~> 7.0'
gem 'pg'

# Install from Gemfile
bundle install
