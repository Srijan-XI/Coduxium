Introduction

What is Lua?

Lua is a compact and embeddable scripting language designed for extension. It is widely used in game engines, plugin systems, and embedded platforms because it is easy to integrate and simple to reason about.

Key Features

Small Runtime - excellent for constrained environments
Simple Syntax - fast ramp-up for teams
Tables - unified data structure
Coroutines - cooperative concurrency
C API - straightforward host integration
Portable - runs across major platforms

User Guide

Basics

local name = "Lua"
print("Hello from " .. name)

for i = 1, 3 do
  print("Tick", i)
end

Tables and Functions

local user = { name = "Ada", level = 5 }

local function promote(u)
  u.level = u.level + 1
end

promote(user)
print(user.level)

Modules

-- modules/mathx.lua
local M = {}

function M.square(x)
  return x * x
end

return M
-- main.lua
local mathx = require("modules.mathx")
print(mathx.square(9))

Advanced Topics

  • Metatables: customize object behavior and operator semantics.
  • Coroutines: implement cooperative task scheduling patterns.
  • Embedding: expose host APIs to Lua for plugins and scriptable workflows.
  • Performance: optimize hot paths and use LuaJIT when compatible.
