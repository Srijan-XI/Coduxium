📚 Introduction

What are C & C++?

C is a procedural programming language created in 1972, forming the foundation of modern operating systems. C++ extends C with object-oriented features, templates, and the STL. Together, they power operating systems, databases, game engines, embedded systems, and performance-critical applications.

Key Features

⚡ Performance — Compiled to native machine code
🎮 Low-Level Access — Direct memory manipulation
🔧 Manual Memory — Full control over allocation
📦 STL (C++) — Standard Template Library
🎯 OOP (C++) — Classes, inheritance, polymorphism
🌐 Portable — Runs on virtually all platforms
⚙️ Systems Level — OS, drivers, firmware
🏎️ Zero Overhead — Pay only for what you use

Why Choose C/C++?

  • Performance: Maximum speed for critical applications
  • Control: Direct hardware and memory access
  • Legacy: Powers Linux, Windows, macOS kernels
  • Game Development: Unreal Engine, Unity (backend)
  • Embedded Systems: IoT, robotics, automotive
  • Foundation: Understanding C/C++ helps with all languages
⚠️ Note: C/C++ require careful memory management. Memory leaks and buffer overflows are common pitfalls for beginners.

Templates (C++)

// Function template
template 
T max(T a, T b) {
    return (a > b) ? a : b;
}

// Class template
template 
class Container {
private:
    T value;
public:
    Container(T v) : value(v) {}
    T getValue() { return value; }
};

int main() {
    std::cout << max(5, 10) << std::endl;
    std::cout << max(3.14, 2.71) << std::endl;
    
    Container intContainer(42);
    Container strContainer("Hello");
    
    return 0;
}
