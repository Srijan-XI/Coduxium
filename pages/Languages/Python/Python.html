📚 Introduction

What is Python?

Python is a high-level, interpreted, dynamically-typed programming language created by Guido van Rossum and first released in 1991. Known for its emphasis on code readability and simplicity, Python has become one of the most popular programming languages worldwide, powering everything from simple scripts to complex web applications and AI systems.

Key Features

📖 Readable Syntax — Clean, English-like code
🔄 Interpreted — No compilation step needed
🎯 Dynamic Typing — Variables don't need type declarations
🧩 Multi-Paradigm — OOP, functional, procedural
📦 Standard Library — "Batteries included" philosophy
🌐 Cross-Platform — Windows, macOS, Linux
🤝 Community — 400,000+ packages on PyPI
⚡ Productivity — Write less code, achieve more

Why Choose Python?

  • Easy to Learn: Intuitive syntax perfect for beginners
  • Versatile: Web development, data science, ML, automation, scripting
  • Rich Ecosystem: NumPy, Pandas, Django, Flask, TensorFlow, PyTorch
  • Job Market: High demand across industries
  • Community Support: Massive, helpful community worldwide
  • Rapid Development: Prototype and deploy quickly
⚠️ Important: Python 2 reached end-of-life in 2020. Always use Python 3.x (3.8+) for new projects.

Functions

Classes & Objects

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return f"Hi, I'm {self.name}"

# Create instance
person = Person("Alice", 30)
print(person.greet())

Decorators

def timer_decorator(func):
    import time
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print(f"Took {time.time() - start:.2f}s")
        return result
    return wrapper

@timer_decorator
def slow_function():
    import time
    time.sleep(1)

Generators

def fibonacci(n):
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

for num in fibonacci(10):
    print(num)
