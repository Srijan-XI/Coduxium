📚 Introduction

What is PHP?

PHP (Hypertext Preprocessor) is a popular server-side scripting language designed for web development. Created in 1994, PHP is embedded in HTML and powers major platforms like WordPress, Drupal, and Laravel applications.

Key Features

🌐 Web-Focused — Built for web development
📝 Easy Syntax — Similar to C, Java, Perl
🔌 Database Integration — MySQL, PostgreSQL, MongoDB
🆓 Open Source — Free and widely supported
🚀 Fast Development — Rapid prototyping
📦 Rich Ecosystem — Composer, frameworks

Why Choose PHP?

  • Ubiquitous: Runs on almost all web hosts
  • WordPress: Powers 43% of all websites
  • Laravel: Modern, elegant PHP framework
  • Easy to Learn: Great for beginners

📖 User Guide

Basic Syntax

<?php
// Variables
$name = "Alice";
$age = 30;
$price = 99.99;

// Echo/Print
echo "Hello, $name";
print "Age: $age";

// Concatenation
$greeting = "Hello, " . $name;

// Arrays
$fruits = ["apple", "banana", "cherry"];
$person = [
    "name" => "Alice",
    "age" => 30
];

echo $fruits[0]; // apple
echo $person["name"]; // Alice
?>

Functions

<?php
function greet($name) {
    return "Hello, $name!";
}

echo greet("Alice");

// Default parameters
function power($base, $exponent = 2) {
    return pow($base, $exponent);
}

// Arrow functions (PHP 7.4+)
$multiply = fn($a, $b) => $a * $b;
?>

Classes & Objects

<?php
class Person {
    private $name;
    private $age;
    
    public function __construct($name, $age) {
        $this->name = $name;
        $this->age = $age;
    }
    
    public function greet() {
        return "Hi, I'm {$this->name}";
    }
    
    public function getName() {
        return $this->name;
    }
}

$person = new Person("Alice", 30);
echo $person->greet();
?>

Database (MySQL)

<?php
// PDO (recommended)
$pdo = new PDO("mysql:host=localhost;dbname=mydb", "user", "password");

// Fetch data
$stmt = $pdo->query("SELECT * FROM users");
while ($row = $stmt->fetch()) {
    echo $row['name'];
}

// Prepared statements (safe)
$stmt = $pdo->prepare("SELECT * FROM users WHERE id = ?");
$stmt->execute([$userId]);

?>

Forms & Superglobals

<?php
// GET
$name = $_GET['name'] ?? 'Guest';

// POST
$email = $_POST['email'] ?? '';

// SERVER
$method = $_SERVER['REQUEST_METHOD'];

// Form handling
if ($_SERVER['REQUEST_METHOD'] === 'POST') {
    $name = htmlspecialchars($_POST['name']);
    $email = filter_var($_POST['email'], FILTER_VALIDATE_EMAIL);
}
?>
