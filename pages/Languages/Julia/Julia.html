Introduction

What is Julia?

Julia is a dynamic language created for scientific and technical computing. It combines expressive syntax with a JIT-compiled execution model and multiple dispatch, making it a strong choice for research and production pipelines.

Key Features

Multiple Dispatch - flexible method specialization
JIT Compilation - high runtime performance
Array-first Design - natural numerical computing
Package Manager - reproducible project environments
Interoperability - call C, Python, and more
Parallelism - multi-threading and distributed support

Installation & Setup

# Windows
winget install julia -s winget

# macOS
brew install julia

# Linux
sudo snap install julia --classic
julia --version
julia

Create a Project Environment

mkdir julia-demo
cd julia-demo
julia --project=.

using Pkg
Pkg.activate(".")
Pkg.add("DataFrames")
Pkg.add("Plots")

User Guide

Hello Julia

println("Hello from Julia!")

Functions and Multiple Dispatch

area(r::Float64) = pi * r^2
area(w::Int, h::Int) = w * h

println(area(3.0))
println(area(5, 2))

DataFrames Quick Start

using DataFrames

df = DataFrame(name=["A", "B"], score=[90, 95])
println(df)
println(mean(df.score))
