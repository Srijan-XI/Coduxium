📚 Introduction

What is R?

R is a programming language and environment for statistical computing and graphics. Created by statisticians for statisticians, R excels at data manipulation, statistical modeling, and creating publication-quality visualizations.

Key Features

📈 Statistics — Built for statistical analysis
📊 Visualization — ggplot2, plotly
📦 CRAN — 19,000+ packages
🔬 Research — Academic standard
🤖 ML/AI — Machine learning libraries
📝 R Markdown — Reproducible reports

Why Choose R?

  • Statistics: Comprehensive statistical functions
  • Visualization: Beautiful, publication-ready plots
  • Community: Strong academic and industry support
  • Free & Open Source: No licensing costs

⚙️ Installation & Setup

Install R

Windows/macOS/Linux

# Download from CRAN
# https://cran.r-project.org/

# Windows
winget install RProject.R

# macOS
brew install r

# Ubuntu/Debian
sudo apt install r-base

# Verify
R --version

Install RStudio (Recommended IDE)

# Download from posit.co
# https://posit.co/download/rstudio-desktop/

# Or use package managers
# Windows
winget install Posit.RStudio

# macOS
brew install --cask rstudio

Your First R Program

# hello.R
print("Hello, R!")

# Run in terminal
Rscript hello.R

# Or in R console
R
> print("Hello, R!")

📖 User Guide

Basic Syntax

# Variables
name <- "Alice"
age <- 30
price <- 99.99

# Vectors
numbers <- c(1, 2, 3, 4, 5)
fruits <- c("apple", "banana", "cherry")

# Basic operations
sum(numbers)
mean(numbers)
median(numbers)
sd(numbers)  # Standard deviation

# Indexing (1-based!)
numbers[1]  # First element
numbers[2:4]  # Range

Data Frames

# Create data frame
df <- data.frame(
  name = c("Alice", "Bob", "Charlie"),
  age = c(30, 25, 35),
  salary = c(70000, 60000, 80000)
)

# View data
head(df)
str(df)
summary(df)

# Access columns
df$name
df[["age"]]

# Filter rows
df[df$age > 25, ]

# Add column
df$bonus <- df$salary * 0.1

Visualization with ggplot2

# Install ggplot2
install.packages("ggplot2")

library(ggplot2)

# Basic plot
ggplot(df, aes(x = age, y = salary)) +
  geom_point() +
  geom_smooth(method = "lm") +
  labs(title = "Age vs Salary",
       x = "Age",
       y = "Salary") +
  theme_minimal()

# Bar plot
ggplot(df, aes(x = name, y = salary, fill = name)) +
  geom_bar(stat = "identity") +
  theme_minimal()

dplyr for Data Manipulation

# Install dplyr
install.packages("dplyr")

library(dplyr)

# Filter
df %>% filter(age > 25)

# Select columns
df %>% select(name, salary)

# Mutate (add/modify columns)
df %>% mutate(bonus = salary * 0.1)

# Group by and summarize
df %>%
  group_by(department) %>%
  summarize(avg_salary = mean(salary))

# Chain operations
df %>%
  filter(age > 25) %>%
  select(name, salary) %>%
  arrange(desc(salary))

Statistical Analysis

# Linear regression
model <- lm(salary ~ age, data = df)
summary(model)

# Predictions
predict(model, newdata = data.frame(age = 28))

# Correlation
cor(df$age, df$salary)

# T-test
t.test(df$salary ~ df$gender)

# ANOVA
aov_result <- aov(salary ~ department, data = df)
summary(aov_result)
