📚 Introduction

What is KQL?

KQL (Kusto Query Language) is a powerful query language for exploring large structured, semi-structured, and unstructured data. Used in Azure Data Explorer, Application Insights, and Log Analytics, KQL helps analyze logs, metrics, and telemetry.

Key Features

📊 Analytics — Fast data exploration
🔍 Search — Full-text and pattern matching
📈 Aggregation — Powerful summarization
⏱️ Time-Series — Time-based analysis
📉 Visualization — Built-in charting
☁️ Cloud-Native — Optimized for Azure

Use Cases

  • Log Analysis: Application and infrastructure logs
  • Performance Monitoring: APM and metrics
  • Security: Threat detection and investigation
  • Business Intelligence: Data exploration and reporting

📖 User Guide

Basic Query Structure

// Table | operator | operator | ...
StormEvents
| where State == "TEXAS"
| take 10

// Comments start with //

Common Operators

where (Filter)

StormEvents
| where State == "CALIFORNIA"
| where EventType == "Flood"
| where DamageProperty > 1000

// Multiple conditions
StormEvents
| where State == "TEXAS" and EventType == "Tornado"

project (Select Columns)

StormEvents
| project State, EventType, DamageProperty

// Rename columns
StormEvents
| project Location=State, Event=EventType, Damage=DamageProperty

// Computed columns
StormEvents
| project State, EventType, DamageInK = DamageProperty / 1000

summarize (Aggregate)

// Count by State
StormEvents
| summarize count() by State

// Multiple aggregations
StormEvents
| summarize 
    TotalEvents = count(),
    TotalDamage = sum(DamageProperty),
    AvgDamage = avg(DamageProperty)
    by State

// Top States by damage
StormEvents
| summarize TotalDamage = sum(DamageProperty) by State
| top 10 by TotalDamage desc

extend (Add Columns)

StormEvents
| extend DamageCategory = case(
    DamageProperty > 1000000, "High",
    DamageProperty > 100000, "Medium",
    "Low"
)

// Add calculated field
StormEvents
| extend DamageInMillions = DamageProperty / 1000000

sort/order

StormEvents
| sort by DamageProperty desc

// Multiple columns
StormEvents
| order by State asc, DamageProperty desc

Time-Series Queries

// Last 24 hours
StormEvents
| where StartTime > ago(24h)

// Specific time range
StormEvents
| where StartTime between (datetime(2024-01-01) .. datetime(2024-01-31))

// Time aggregation
StormEvents
| summarize count() by bin(StartTime, 1d)

// Trend over time
StormEvents
| where StartTime > ago(30d)
| summarize TotalDamage = sum(DamageProperty) by bin(StartTime, 1d)
| render timechart

String Operations

// Contains
StormEvents
| where EventNarrative contains "flood"

// Starts with
StormEvents
| where State startswith "C"

// Regular expressions
StormEvents
| where EventNarrative matches regex "tornado.*damage"

// String functions
StormEvents
| extend UpperState = toupper(State)
| extend EventLength = strlen(EventNarrative)

🚀 Advanced Topics

Joins

// Inner join
StormEvents
| join kind=inner (
    StormDamages
) on EventId

// Left outer join
StormEvents
| join kind=leftouter (
    StormDamages
) on EventId

Visualization

// Pie chart
StormEvents
| summarize count() by State
| render piechart

// Bar chart
StormEvents
| summarize TotalDamage = sum(DamageProperty) by EventType
| render barchart

// Time chart
StormEvents
| summarize count() by bin(StartTime, 1d)
| render timechart

// Scatter plot
StormEvents
| project DamageProperty, DeathsDirect
| render scatterchart

Common Scenarios

Error Analysis

// Application errors
AppTraces
| where SeverityLevel >= 3  // Error or Critical
| where TimeGenerated > ago(1h)
| summarize count() by Message
| top 10 by count_

Performance Monitoring

// Slow requests
AppRequests
| where Duration > 1000  // > 1 second
| summarize 
    AvgDuration = avg(Duration),
    P95Duration = percentile(Duration, 95)
    by Name
