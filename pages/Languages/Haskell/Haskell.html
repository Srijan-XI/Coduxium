Introduction

What is Haskell?

Haskell is a statically typed and purely functional language that emphasizes immutable data, composable abstractions, and high confidence through compile-time verification.

Key Features

Purity - predictable function behavior
Type Inference - expressive code with fewer annotations
ADTs - robust domain modeling
Type Classes - ad-hoc polymorphism
Laziness - demand-driven evaluation
Strong Tooling - GHC, Cabal, Stack, HLS

Installation & Setup

Install via GHCup

Use the official installer to manage compiler, build tools, and language server in one flow.

# Visit https://www.haskell.org/ghcup/
# Install GHCup and select GHC + Cabal + HLS

ghc --version
cabal --version
haskell-language-server-wrapper --version

Create a New Project

mkdir haskell-demo
cd haskell-demo
cabal init
cabal build
cabal run

User Guide

Hello World

module Main where

main :: IO ()

Functions and Pattern Matching

factorial :: Integer -> Integer
factorial 0 = 1
factorial n = n * factorial (n - 1)

describe :: Int -> String
describe 0 = "zero"
describe _ = "non-zero"

Algebraic Data Types

data Role = Admin | Editor | Viewer

canWrite :: Role -> Bool
canWrite Admin = True
canWrite Editor = True
canWrite Viewer = False
