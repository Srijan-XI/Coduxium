NixOS

NixOS is a Linux distribution built around declarative system configuration and reproducible package management with the Nix language.

Overview

Linux Declarative Reproducible Nix

NixOS enables immutable-style workflows where infrastructure and host state can be versioned, reviewed, and rolled back safely.

Key Concepts

  • Declarative system config in configuration.nix
  • Atomic upgrades and rollbacks via generations
  • Reproducible builds and isolated package environments
  • Flakes for composable project and system definitions

Getting Started

# Rebuild with updated config
sudo nixos-rebuild switch

# List generations
sudo nix-env --list-generations --profile /nix/var/nix/profiles/system
