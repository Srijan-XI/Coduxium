📚 Introduction

What is Arch Linux?

Arch Linux is an independently developed, x86-64 general-purpose Linux distribution that strives to provide the latest stable versions of most software by following a rolling-release model. The default installation is a minimal base system, configured by the user to only add what is purposely required. Arch follows the KISS principle (Keep It Simple, Stupid) and focuses on elegance, code correctness, and minimalism.

Key Features

  • Rolling Release: Continuous updates, always latest software versions
  • Pacman: Fast, powerful package manager with dependency resolution
  • AUR (Arch User Repository): Massive community-driven package repository
  • Arch Wiki: Comprehensive documentation covering everything
  • Customization: Build your system exactly how you want it

⚙️ Installation Guide

Step 1: Download & Boot

Download from: archlinux.org/download

Create bootable USB with dd or Etcher
Boot from USB and login as root

Step 2: Partition & Format

# Partition disk
cfdisk /dev/sda

# Format partitions
mkfs.ext4 /dev/sda1  # Root
mkswap /dev/sda2     # Swap
mkfs.ext4 /dev/sda3  # Home

# Mount
mount /dev/sda1 /mnt
mount --mkdir /dev/sda3 /mnt/home
swapon /dev/sda2

Step 3: Install Base System

# Install base packages
pacstrap -K /mnt base linux linux-firmware

# Generate fstab
genfstab -U /mnt >> /mnt/etc/fstab

# Chroot into system
arch-chroot /mnt

Step 4: Configure & Boot

# Set timezone
ln -sf /usr/share/zoneinfo/Region/City /etc/localtime
hwclock --systohc

# Set locale
echo "en_US.UTF-8 UTF-8" >> /etc/locale.gen
locale-gen

# Set hostname
echo "myhostname" > /etc/hostname

# Install bootloader
pacman -S grub
grub-install /dev/sda
grub-mkconfig -o /boot/grub/grub.cfg

# Set root password
passwd

# Reboot
exit
reboot
⚠️ System Requirements
  • CPU: x86_64 compatible (64-bit)
  • RAM: 512MB minimum (2GB+ recommended)
  • Storage: 2GB minimum (20GB+ recommended)
  • GPU: Any (depends on desktop environment choice)

📖 User Guide

Package Management with Pacman

Update System

sudo pacman -Syu

Install Package

sudo pacman -S package-name

# Install multiple
sudo pacman -S firefox vlc gimp

Remove Package

sudo pacman -R package-name

# Remove with dependencies
sudo pacman -Rs package-name

Search Package

pacman -Ss keyword
pacman -Si package-name  # Details

Essential Tasks

Install Desktop Environment

# GNOME
sudo pacman -S gnome gnome-extra

# KDE Plasma
sudo pacman -S plasma kde-applications

# Xfce
sudo pacman -S xfce4 xfce4-goodies

Install Display Manager

sudo pacman -S gdm  # For GNOME
sudo systemctl enable gdm
sudo systemctl start gdm

Install AUR Helper (yay)

sudo pacman -S --needed git base-devel
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si

Using AUR with yay

yay -S package-name
yay -Syu  # Update AUR packages

🚀 Advanced Topics

Kernel Management

# Install LTS kernel
sudo pacman -S linux-lts linux-lts-headers

# Install zen kernel (performance)
sudo pacman -S linux-zen linux-zen-headers

# Update bootloader
sudo grub-mkconfig -o /boot/grub/grub.cfg

Multiple kernel support for stability/performance

System Cleanup

# Remove orphaned packages
sudo pacman -Rns $(pacman -Qtdq)

# Clean package cache
sudo pacman -Sc

# Clean all cache
sudo pacman -Scc

Keep system clean and optimized

Reflector (Mirror List)

# Install reflector
sudo pacman -S reflector

# Update mirrors
sudo reflector --country 'United States' --age 12 --protocol https --sort rate --save /etc/pacman.d/mirrorlist

Optimize download speeds with best mirrors

Timeshift Backups

# Install timeshift
sudo pacman -S timeshift

# Create snapshot
sudo timeshift --create --comments "Before update"

# Restore snapshot
sudo timeshift --restore

System snapshots for easy recovery

💻 Useful Commands

Task Command Description
Full System Update sudo pacman -Syu Sync and update all packages
Install Package sudo pacman -S package Install a package
Remove Package sudo pacman -Rs package Remove package with dependencies
Search Packages pacman -Ss keyword Search for packages
List Installed pacman -Q List all installed packages
Package Info pacman -Si package Show package information
Clean Cache sudo pacman -Sc Remove old package cache
List Orphans pacman -Qtdq Show orphaned packages
File Ownership pacman -Qo /path/to/file Find which package owns a file
Download Only sudo pacman -Sw package Download without installing
