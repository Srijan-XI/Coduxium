📚 Introduction

What is Garuda Linux?

Garuda is an Arch-based rolling-release distribution designed to be approachable for desktop users while keeping the power of the Arch ecosystem. It often ships with Btrfs snapshots to make rolling updates safer.

Key Features

  • Arch base with rolling updates
  • Btrfs snapshots for rollback
  • Desktop-first experience
  • Gaming setup support
  • Multiple desktop editions

Why Choose Garuda Linux?

1

Safer Rolling Updates

Snapshots help recover quickly if an update breaks something.

2

Great Desktop UX

Focus on polish and a ready-to-use desktop.

3

Arch Ecosystem

Access Pacman repos and community packages (AUR).

⚙️ Installation Guide

Step 1: Download

Download from: https://garudalinux.org/downloads

Step 2: Create Bootable USB

# Linux
sudo dd if=garuda.iso of=/dev/sdX bs=4M status=progress && sync

# Or use: Etcher, Rufus, Ventoy

Step 3: Boot from USB

Boot the live environment and verify graphics/network.

Step 4: Install

Run the graphical installer, choose Btrfs (recommended by Garuda), and complete the install.

⚠️ System Requirements
  • CPU: 64-bit
  • RAM: 4GB minimum (8GB+ recommended)
  • Storage: 30GB+ recommended
  • GPU: Any modern GPU; proprietary drivers optional

Package Management
