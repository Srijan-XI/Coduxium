Garuda Linux

Arch Linux made easy, beautiful, and fast.

🦅 Overview

Garuda Linux is an Arch-based Rolling Release distro that focuses on performance and ease of use. It uses the Zen Kernel and BTRFS file system by default.

🎮 Gaming Specs

Known for its "Dragonized" gaming edition, Garuda comes pre-configured with gaming tools (Steam, Lutris, GameMode) and optimized drivers.

💾 BTRFS & Snapper

Automatic snapshots before every update ensure you can always roll back your system if something breaks. A true lifesaver for rolling release users.
