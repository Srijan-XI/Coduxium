📚 Introduction

What is RedArch?

RedArch is an Arch-based security distribution intended for penetration testing and red team operations. It builds on Arch's package ecosystem and supports typical security tooling setups.

Key Features

  • Arch base and Pacman workflow
  • Security and red team focus
  • Rolling updates
  • Flexible desktop choices
  • Works well in VM/lab environments

⚙️ Installation Guide

Step 1: Download

If an official download is available, verify checksums and use the project site. If not, consider Arch + selected tools instead.

Step 2: Create Bootable USB

# Linux
sudo dd if=redarch.iso of=/dev/sdX bs=4M status=progress && sync

Step 3: Boot from USB

Boot the live environment and ensure network and graphics are working.

Step 4: Install

Use the installer (if provided) or install on top of Arch using Pacman and configured repositories.

⚠️ System Requirements
  • CPU: 64-bit
  • RAM: 2GB minimum (4GB+ recommended)
  • Storage: 20GB+ recommended
  • GPU: Any compatible with your desktop

📖 User Guide

Package Management

Update System

sudo pacman -Syu

Install Package

sudo pacman -S package-name

Remove Package

sudo pacman -Rns package-name

Search Package

pacman -Ss keyword
