BlackArch Linux

The Arch Linux-based penetration testing distribution.

🐱‍💻 Overview

BlackArch is intended for penetration testers and security researchers. The repository contains 2800+ tools that can be installed individually or in groups.

⚙️ Installation

Can be installed as a complete ISO or added as an unofficial user repository to an existing Arch Linux installation.

# Add repo to pacman.conf
[blackarch]
SigLevel = Optional TrustAll
Server = https://blackarch.org/blackarch/$repo/os/$arch
