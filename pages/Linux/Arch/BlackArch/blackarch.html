📚 Introduction

What is BlackArch?

BlackArch is a penetration testing distribution and repository built on Arch Linux. You can run it as a full OS or add the BlackArch repository to a standard Arch install to pull security tools as needed.

Key Features

  • Large security tool repository
  • Install tools individually or by category
  • Rolling updates via Arch ecosystem
  • Minimal base is possible (build your own environment)
  • Works as ISO or as an added repository

Why Choose BlackArch?

1

Tool Coverage

Access a wide range of offensive and defensive tooling.

2

Selective Installs

Install only what you need to keep systems lean.

3

Arch Workflow

Familiar Pacman + rolling release cadence.

⚙️ Installation Guide

Step 1: Download

Download from: https://blackarch.org/downloads.html

Step 2: Create Bootable USB

# Linux
sudo dd if=blackarch.iso of=/dev/sdX bs=4M status=progress && sync

# Or use: Etcher, Rufus, Ventoy

Step 3: Boot from USB

Boot and choose live session or installer depending on ISO.

Step 4: Add Repo (Optional)

# Add BlackArch repo to existing Arch
# Follow official instructions on blackarch.org

sudo pacman -Syu
⚠️ System Requirements
  • CPU: 64-bit
  • RAM: 2GB minimum (4GB+ recommended)
  • Storage: 20GB+ recommended
  • GPU: Any compatible with your chosen desktop

📖 User Guide

Package Management

Update System

sudo pacman -Syu

Install Package

sudo pacman -S package-name

Remove Package

sudo pacman -Rns package-name

Search Package

pacman -Ss keyword

Essential Tasks

Enable Firewall

Use UFW/nftables depending on your setup.

Keep Tools Isolated

Use VMs/containers for risky tooling and testing.

Snapshots

Use filesystem snapshots before big upgrades.

Update Mirrors

Use reflector to pick fast mirrors (if installed).

🚀 Advanced Topics

Pacman Mirror Refresh

sudo reflector --latest 20 --sort rate --save /etc/pacman.d/mirrorlist

Optional, improves download performance.
