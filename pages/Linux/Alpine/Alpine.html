📚 Introduction

What is Alpine Linux?

Alpine Linux is a security-oriented, lightweight Linux distribution designed for power users who appreciate security, simplicity, and resource efficiency. Unlike traditional distributions that use glibc, Alpine uses musl libc and BusyBox to achieve an incredibly small footprint. This makes it ideal for containers, cloud deployments, embedded systems, and any environment where resources are limited.

Key Features

  • Tiny Size: Base Docker image is only ~5MB (compared to 100MB+ for others)
  • Security Hardened: Uses PaX and grsecurity patches for enhanced security
  • musl libc: Lightweight, standards-compliant C library
  • BusyBox: Combines hundreds of Unix utilities into single executable
  • APK Package Manager: Simple, fast package management system

⚙️ Installation Guide

Step 1: Download

Download Alpine Linux from: alpinelinux.org/downloads

Choose: Standard (~150MB), Extended (desktop), Virtual, or NetBoot

Step 2: Create Bootable USB

# Linux
sudo dd if=alpine-standard.iso of=/dev/sdX bs=4M

# Or use: Etcher, Rufus, Ventoy

Step 3: Boot from USB

1. Insert USB drive
2. Restart computer
3. Enter BIOS (F2/F12/Del)
4. Select USB as boot device
5. Login as root (no password)

Step 4: Run Setup

# Run Alpine setup script
setup-alpine

# Follow prompts for:
# - Keyboard layout
# - Hostname, network
# - Root password, timezone
# - APK mirror, SSH
# - Disk and installation mode

# Reboot after completion
reboot
⚠️ System Requirements
  • CPU: x86_64, ARM, or other (very flexible)
  • RAM: 128MB minimum (512MB recommended)
  • Storage: 700MB minimum (2GB recommended)
  • GPU: Not required (text-based by default)

📖 User Guide

Package Management with APK

Update System

apk update
apk upgrade

Install Package

apk add package-name

# Install without caching (for Docker)
apk add --no-cache package-name

Remove Package

apk del package-name

Search Package

apk search keyword
apk search -v -d keyword  # Verbose with description

Essential Tasks

Install Common Tools

apk add bash curl wget git vim nano

Service Management (OpenRC)

rc-service servicename start
rc-update add servicename default

Install Desktop (Xfce)

setup-xorg-base
apk add xfce4 xfce4-terminal lightdm
rc-update add lightdm

List Installed Packages

apk info
apk info -vv  # Detailed info

🚀 Advanced Topics

Docker Multi-Stage Build

FROM alpine:latest AS builder
RUN apk add --no-cache go
COPY . /app
WORKDIR /app
RUN go build -o myapp

FROM alpine:latest
RUN apk add --no-cache ca-certificates
COPY --from=builder /app/myapp /usr/local/bin/
CMD ["myapp"]

Optimize Docker images with Alpine's small size

Firewall Configuration

# Install awall firewall
apk add awall iptables
rc-update add iptables

# Configure and enable
vi /etc/awall/optional/custom.json
awall enable custom
awall activate

Set up Alpine Wall firewall frontend

Development Environment

# Install build tools
apk add build-base

# Programming languages
apk add python3 py3-pip nodejs npm rust

# Database
apk add postgresql mariadb

Set up development tools and languages

Custom Package Building

# Install build tools
apk add alpine-sdk

# Add user for building
adduser -D builder
addgroup builder abuild

# Build package
abuild -r

Build custom APK packages from source
