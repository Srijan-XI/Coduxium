📚 Introduction

What is Ubuntu?

Ubuntu is a Debian-based Linux distribution developed by Canonical Ltd. Since its launch in 2004, Ubuntu has become the most widely used Linux distribution for desktops and is also popular for servers and cloud deployments. The name "Ubuntu" comes from an African philosophy meaning "humanity towards others." Ubuntu emphasizes ease of use, regular releases, and a strong community focus.

Key Features

  • Modern GNOME Desktop: Beautiful, intuitive interface with excellent usability
  • LTS Releases: Long Term Support versions receive 5 years of updates
  • Snap Packages: Universal package format with automatic updates and sandboxing
  • Ubuntu Software: Easy-to-use app store for discovering and installing software
  • Hardware Compatibility: Excellent out-of-the-box support for most hardware

⚙️ Installation Guide

Step 1: Download

Download Ubuntu from: ubuntu.com/download

Choose: Ubuntu Desktop (latest) or Ubuntu LTS (long-term support)
File size: ~3-4GB

Step 2: Create Bootable USB

# Linux
sudo dd if=ubuntu.iso of=/dev/sdX bs=4M status=progress

# Or use: Etcher, Rufus (Windows), Ventoy

Step 3: Boot from USB

1. Insert USB drive
2. Restart computer
3. Enter BIOS/UEFI (F2/F12/Del)
4. Select USB as boot device
5. Choose "Try or Install Ubuntu"

Step 4: Install Ubuntu

1. Click "Install Ubuntu"
2. Choose language and keyboard
3. Select "Normal installation"
4. Choose disk partitioning
5. Create user account
6. Wait for installation
7. Restart and enjoy!

⚠️ System Requirements
  • CPU: 2 GHz dual-core or better
  • RAM: 4GB minimum (8GB recommended)
  • Storage: 25GB minimum (50GB+ recommended)
  • GPU: Graphics card with 1024x768 resolution

📖 User Guide

Package Management with APT

Update System

sudo apt update
sudo apt upgrade -y

Install Package

sudo apt install package-name

# Install multiple packages
sudo apt install firefox vlc gimp

Remove Package

sudo apt remove package-name

# Remove with config files
sudo apt purge package-name

Search Package

apt search keyword
apt show package-name  # Details

Essential Tasks

Ubuntu Software Center

Click "Show Applications" → "Ubuntu Software" to browse and install apps graphically

Install Snap Packages

snap install package-name
snap find keyword

System Settings

Access Settings from top-right menu to configure display, network, users, and more

File Manager (Nautilus)

Press Super key → type "Files" or click Files icon in dock

🚀 Advanced Topics

Enable Additional Drivers

# Open driver manager
software-properties-gtk

# Or via command line
ubuntu-drivers devices
sudo ubuntu-drivers autoinstall

Install proprietary drivers for NVIDIA, Wi-Fi, etc.

Add PPAs (Personal Package Archives)

# Add PPA
sudo add-apt-repository ppa:user/ppa-name
sudo apt update

# Remove PPA
sudo add-apt-repository --remove ppa:user/ppa-name

Access additional software repositories

System Cleanup

# Remove unused packages
sudo apt autoremove

# Clean package cache
sudo apt clean

# Remove old kernels (careful!)
sudo apt remove --purge $(dpkg -l 'linux-*' | grep ^ii | awk '{print $2}' | grep -v `uname -r`)

Free up disk space

Install Development Tools

# Build essentials
sudo apt install build-essential git

# Programming languages
sudo apt install python3-pip nodejs npm

# IDEs
sudo snap install code --classic  # VS Code
sudo snap install pycharm-community --classic

Set up development environment

💻 Useful Commands

Task Command Description
Update System sudo apt update && sudo apt upgrade Update package list and upgrade all packages
Install Software sudo apt install package Install a new package
Remove Software sudo apt remove package Uninstall a package
Search Packages apt search keyword Search for available packages
Clean System sudo apt autoremove && sudo apt clean Remove unused packages and cache
System Info lsb_release -a Show Ubuntu version information
Disk Usage df -h Check disk space usage
Running Processes top or htop Monitor system processes
Network Info ip addr or ifconfig Show network configuration
Restart Services sudo systemctl restart service-name Restart a system service
