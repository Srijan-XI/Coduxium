📚 Introduction

What is Parrot Security?

Parrot Security OS is a Debian-based distribution built for security professionals and privacy-minded users. It includes a curated set of tools for penetration testing, forensics, and development, while aiming to stay lightweight.

Key Features

  • Security tooling for auditing and testing
  • Privacy-focused utilities and workflows
  • Lightweight desktop experience
  • Debian base for broad compatibility
  • Good for labs, training, and learning environments

Why Choose Parrot Security?

1

Start practicing security tasks quickly without assembling a toolkit from scratch.

2

Debian Compatibility

3

Lightweight UX

Useful on laptops and VMs where resources are limited.

⚙️ Installation Guide

Step 1: Download

Download Parrot OS from: https://www.parrotsec.org/

Step 2: Create Bootable USB

# Linux
sudo dd if=Parrot.iso of=/dev/sdX bs=4M status=progress && sync

# Or use: Etcher, Rufus, Ventoy

Step 3: Boot from USB

1. Insert USB drive
2. Restart computer
3. Enter BIOS/UEFI (F2/F12/Del)
4. Select USB as boot device

Step 4: Installation Process

Start the live environment (if available), launch the installer, select language/keyboard, configure disk and users, then complete installation and reboot.

⚠️ System Requirements
  • CPU: 64-bit compatible
  • RAM: 2GB minimum (4GB+ recommended)
  • Storage: 20GB+ recommended
  • GPU: Any (depends on desktop)
