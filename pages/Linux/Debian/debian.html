📚 Introduction

What is Debian?

Debian is one of the oldest and most influential Linux distributions. It focuses on stability, free software principles, and a strong packaging system. Many downstream distributions (like Ubuntu) build on Debian’s work.

Key Features

  • Stable releases and conservative change management
  • Large package repositories and mature tooling
  • Multiple supported desktops and server-first options
  • Strong community documentation and ecosystem
  • Supports many CPU architectures

Why Choose Debian?

1

Stability First

Great for servers and long-running systems.

2

Trusted Base

Downstream distros inherit Debian’s packaging foundation.

3

Flexibility

Choose stable/testing and your preferred desktop stack.

⚙️ Installation Guide

Step 1: Download

Download from: https://www.debian.org/distrib/

Step 2: Create Bootable USB

# Windows: Rufus
# macOS/Linux: Etcher

# Linux (dd example)
sudo dd if=debian.iso of=/dev/sdX bs=4M status=progress && sync

Step 3: Boot Installer

Boot from USB and choose Installer / Graphical Installer.

Step 4: Install

Select disk options, choose a desktop environment, and complete setup.

⚠️ Notes
  • Some Wi‑Fi chipsets may need non-free firmware packages.
  • For newer hardware, consider the latest stable point release or testing.

📖 User Guide

Package Management

Update System

sudo apt update
sudo apt upgrade

Install Software

sudo apt install package-name

Remove Software

sudo apt remove package-name
sudo apt autoremove

Search

apt search keyword

Essential Tasks

Users

Create a non-root user and use sudo for admin tasks.

Updates

Enable security updates and keep packages current.

Firewall

Use UFW or nftables for basic host firewalling.

Backups

Schedule backups for /home and key system configs.

🚀 Advanced Topics

Enable Backports

# Example: add stable-backports (adjust to your release)
echo "deb http://deb.debian.org/debian stable-backports main" | sudo tee /etc/apt/sources.list.d/backports.list
sudo apt update

Get newer packages while staying on stable.

Firewall (UFW)

sudo apt install ufw
sudo ufw enable
sudo ufw status

Simple firewall management.

Service Debugging

systemctl status service-name
journalctl -u service-name --no-pager -n 200

Troubleshoot systemd services.

Upgrade Strategy

# Before a major upgrade
sudo apt update
sudo apt full-upgrade

Keep the system consistent before release upgrades.
