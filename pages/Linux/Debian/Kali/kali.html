📚 Introduction

What is Kali Linux?

Kali Linux is a Debian-based Linux distribution designed for digital forensics and penetration testing. Maintained and funded by Offensive Security, Kali Linux provides a comprehensive platform for security professionals with over 600 pre-installed penetration testing tools. It's the go-to operating system for security audits, penetration testing, and ethical hacking.

Key Features

  • Metasploit Framework: Comprehensive penetration testing framework
  • Nmap: Network discovery and security auditing
  • Wireshark: Network protocol analyzer
  • Aircrack-ng: Wireless network security testing suite
  • Burp Suite: Web application security testing

⚙️ Installation Guide

Step 1: Download

Download from: kali.org/get-kali

Choose: Installer, Live, or Virtual Machine image
File size: ~3-4GB

Step 2: Create Bootable USB

# Linux
sudo dd if=kali-linux.iso of=/dev/sdX bs=4M status=progress

# Or use: Etcher, Rufus, Win32DiskImager

Step 3: Boot from USB

1. Insert USB drive
2. Restart computer
3. Enter BIOS/UEFI (F2/F12/Del)
4. Select USB as boot device
5. Choose "Graphical Install"

Step 4: Install Kali

1. Select language and location
2. Configure network
3. Set up users and passwords
4. Partition disks
5. Select software (default is fine)
6. Install GRUB bootloader
7. Reboot and login

⚠️ System Requirements
  • CPU: 2 GHz dual-core or better
  • RAM: 2GB minimum (4GB+ recommended for tools)
  • Storage: 20GB minimum (50GB+ recommended)
  • GPU: Graphics card with 1024x768 resolution

📖 User Guide

Package Management with APT

Update System

sudo apt update
sudo apt full-upgrade -y

Install Tools

sudo apt install tool-name

# Install metapackages
sudo apt install kali-tools-top10

Remove Package

sudo apt remove package-name
sudo apt autoremove

Search Tools

apt search keyword
apt show package-name

Essential Security Tools

Network Scanning (Nmap)

# Basic scan
nmap target-ip

# Detailed scan
nmap -A -T4 target-ip

Metasploit Framework

# Start Metasploit
msfconsole

# Update Metasploit
sudo msfupdate

Wireless Testing (Aircrack-ng)

# Monitor mode
sudo airmon-ng start wlan0

# Capture packets
sudo airodump-ng wlan0mon

Web Testing (Burp Suite)

# Start Burp Suite
burpsuite

# Or from menu
Applications → Web Application Analysis

🚀 Advanced Topics

Install Kali Metapackages

# Top 10 tools
sudo apt install kali-tools-top10

# All tools (large!)
sudo apt install kali-linux-everything

# Web application testing
sudo apt install kali-tools-web

# Wireless testing
sudo apt install kali-tools-wireless

Install tool collections by category

Custom Kali ISO

# Install live-build
sudo apt install git live-build cdebootstrap

# Clone Kali build scripts
git clone https://gitlab.com/kalilinux/build-scripts/live-build-config.git

# Build custom ISO
cd live-build-config
sudo ./build.sh --variant xfce --verbose

Create customized Kali Linux ISO

Kali in Docker

# Pull Kali image
docker pull kalilinux/kali-rolling

# Run Kali container
docker run -it kalilinux/kali-rolling /bin/bash

# Update and install tools
apt update && apt install kali-tools-top10

Run Kali in containerized environment

Non-Root User Setup

# Add new user
sudo useradd -m -s /bin/bash username
sudo passwd username

# Add to sudo group
sudo usermod -aG sudo username

# Switch to user
su - username

Run Kali as non-root for better security
