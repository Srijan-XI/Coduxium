📚 Introduction

What is Pop!_OS?

Pop!_OS is a free and open-source Linux distribution developed by System76, a computer manufacturer specializing in Linux hardware. While based on Ubuntu, Pop!_OS includes significant customizations and improvements focused on productivity, gaming, and professional workflows. It features the COSMIC desktop environment (a heavily customized GNOME), automatic window tiling, and excellent hardware support especially for NVIDIA graphics cards.

Key Features

  • COSMIC Desktop: Custom GNOME-based desktop with productivity enhancements
  • Auto-Tiling: Keyboard-driven window management for power users
  • Graphics Switching: Easy switching between integrated and discrete GPUs
  • Pop!_Shop: Curated app store with Flatpak and Debian packages
  • Recovery Partition: Built-in recovery and refresh capabilities

⚙️ Installation Guide

Step 1: Download

Download from: pop.system76.com

Choose: NVIDIA (for NVIDIA GPUs) or Intel/AMD version
File size: ~2.5GB

Step 2: Create Bootable USB

# Linux
sudo dd if=pop-os.iso of=/dev/sdX bs=4M status=progress

# Or use Popsicle (Pop!_OS USB Creator)
# Or Etcher, Rufus

Step 3: Boot from USB

1. Insert USB drive
2. Restart computer
3. Enter BIOS/UEFI (F2/F12/Del)
4. Select USB as boot device
5. Choose "Try or Install Pop!_OS"

Step 4: Install Pop!_OS

1. Select language
2. Choose keyboard layout
3. Select installation type (Clean/Custom)
4. Configure disk encryption (optional)
5. Create user account
6. Wait for installation
7. Reboot and enjoy!

⚠️ System Requirements
  • CPU: 2 GHz dual-core or better (64-bit)
  • RAM: 4GB minimum (8GB+ recommended)
  • Storage: 20GB minimum (50GB+ recommended)
  • GPU: Intel, AMD, or NVIDIA (separate ISO for NVIDIA)

📖 User Guide

Package Management

Update System

sudo apt update
sudo apt full-upgrade -y

Install via APT

sudo apt install package-name

Install via Flatpak

flatpak install flathub app-name

# Or use Pop!_Shop (GUI)

Remove Package

sudo apt remove package-name
sudo apt autoremove

Essential Features

Auto-Tiling

Enable: Super + Y
Navigate: Super + Arrow Keys
Move windows: Super + Shift + Arrow Keys

Graphics Switching

# Switch to NVIDIA
system76-power graphics nvidia

# Switch to integrated
system76-power graphics integrated

# Hybrid mode
system76-power graphics hybrid

Pop!_Shop

Activities → Pop!_Shop
Browse apps, games, and developer tools
Supports both APT and Flatpak

Recovery Mode

Hold Space at boot → Recovery
Refresh OS (keeps /home)
Reinstall from recovery partition
