📚 Introduction

What is Tails?

Tails (The Amnesic Incognito Live System) is a Debian-based live operating system aimed at preserving privacy and anonymity. It is designed to be used from a USB stick and routes internet connections through Tor, while trying not to leave traces on the host computer.

Key Features

  • Tor Browser with security levels
  • Amnesic shutdown (no traces by default)
  • Optional encrypted persistent storage
  • Encrypted communication tools (e.g., email with OpenPGP)
  • Built-in tools for safer file handling

Why Choose Tails?

1

Stay Anonymous

Use Tor for browsing and communications in a privacy-focused environment.

2

Portable Security

Carry a secure environment on a USB stick for use on many machines.

3

Leave No Trace

Amnesic design reduces local traces after shutdown by default.

⚙️ Installation Guide

Step 1: Download

Download Tails from: https://tails.net/install/

Step 2: Create Bootable USB

# Recommended: follow the official installer instructions

# Linux (example with dd)
sudo dd if=tails-amd64.img of=/dev/sdX bs=16M status=progress && sync

# Or use: Etcher, Rufus, Ventoy

Step 3: Boot from USB

1. Insert USB drive
2. Restart computer
3. Use boot menu (F12/ESC/F2 varies)
4. Select USB to launch Tails

Step 4: First Boot Setup

Configure language/keyboard, optional admin password, and Tor bridges if needed, then start Tails.

⚠️ System Requirements
  • CPU: 64-bit compatible
  • RAM: 2GB minimum (more recommended)
  • Storage: 8GB USB minimum
  • GPU: Any compatible with GNOME
