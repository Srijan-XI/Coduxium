How to Install Linux

You can install Linux as your main OS, alongside Windows (Dual Boot), in a Virtual Machine, or via WSL. This guide covers the general process for most distributions.

Method 1: The "Real" Installation

Best for performance and full hardware access.

1

Download the ISO

Go to the official website of your chosen distro (e.g., Ubuntu, Fedora, Linux Mint) and download the ISO image file.

2

Create Bootable Media

You need a USB drive (8GB+). Use software to "burn" the ISO to the USB.

  • Windows: Use Rufus or BalenaEtcher.
  • macOS/Linux: Use BalenaEtcher or `dd` command.
3

Boot from USB

Plug the USB into your PC. Restart and press the Boot Menu key (F12, F2, Esc, or Del) depending on your PC brand. Select the USB drive.

4

Install

Follow the on-screen wizard.

  • Partitions: "Erase disk and install" is easiest. "Something else" for manual partitioning (usually separate `/` and `/home`).
  • User Setup: Create your username and password.

Method 2: Virtual Machine (Safe & Easy)

Run Linux inside Windows/macOS. No risk of breaking your main system.

  1. Download VirtualBox or VMware Workstation Player.
  2. Create a "New Virtual Machine".
  3. Select the downloaded Linux ISO.
  4. Allocate RAM (4GB+) and Disk Space (20GB+).
  5. Start the VM and follow the installer.

Method 3: Windows Subsystem for Linux (WSL)

🚀 The Developer's Choice

Run a real Linux kernel directly on Windows without a VM.


1. Open PowerShell (Admin)

wsl --install

2. Reboot your computer.

3. Open "Ubuntu" from start menu and set your username.
