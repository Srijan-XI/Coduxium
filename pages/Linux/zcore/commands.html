Essential Linux Commands

A clean, practical cheat sheet for daily terminal work. sudo typically requires admin privileges.

📂 File Operations

Command Description
ls -la List all files (including hidden) with details.
cd directory Change directory. Use cd .. to go back one level.
cp source dest Copy a file. Use cp -r for folders.
mv source dest Move or rename a file.
rm file Remove a file. Dangerous: rm -rf folder deletes a folder.
mkdir folder Create a new directory.
touch file Create an empty file.
cat file Display file content.
grep "text" file Search for text inside a file.

⚡ System Info & Management

Command Description
sudo Run command as superuser (admin).
top / htop Monitor real-time processes and resource usage.
df -h Check disk space usage.
free -h Check RAM usage.
uname -a Show kernel and system information.
systemctl status service Check the status of a service (e.g., nginx, sshd).
shutdown now Turn off the computer immediately.

📦 Package Management

Commands depend on your distribution family.

Family Update Install Remove
Debian/Ubuntu apt update && apt upgrade apt install pkg apt remove pkg
Fedora/RHEL dnf upgrade dnf install pkg dnf remove pkg
Arch Linux pacman -Syu pacman -S pkg pacman -Rns pkg

🔒 Permissions

Command Description
chmod +x file Make a file executable.
chmod 777 file Give everyone read/write/execute permissions (risky).
chown user:group file Change file ownership.
