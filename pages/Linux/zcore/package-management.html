📚 Introduction

💡 What is a Package Manager?

A package manager is a collection of software tools that automates the process of installing, upgrading, configuring, and removing software packages from your Linux system.

🔄

Automatic Dependencies

Automatically resolves and installs required dependencies

🔐

Security

Verifies package authenticity and integrity

⚡

Updates

Keeps your system and software up-to-date

🗃️

Repositories

Access to thousands of pre-built packages

🔧 Package Managers by Distribution

APT - Advanced Package Tool

Debian Ubuntu Mint Pop!_OS

Update Package Lists

sudo apt update

Refreshes the list of available packages

Upgrade All Packages

sudo apt upgrade

Upgrades all installed packages

Full System Upgrade

sudo apt full-upgrade

Upgrades with dependency handling

Install Package

sudo apt install package-name

Installs a specific package

Remove Package

sudo apt remove package-name

Removes a package (keeps config)

Purge Package

sudo apt purge package-name

Completely removes package and config

Search Packages

apt search keyword

Searches for packages

Show Package Info

apt show package-name

Displays detailed package information

Clean Cache

sudo apt autoclean
sudo apt autoremove

Removes unnecessary packages and cache

DNF - Dandified YUM

Fedora RHEL Rocky AlmaLinux

Update Package Cache

sudo dnf check-update

Checks for available updates

Upgrade All Packages

sudo dnf upgrade

Updates all packages

Install Package

sudo dnf install package-name

Installs a package

Remove Package

sudo dnf remove package-name

Removes a package

Search Packages

dnf search keyword

Searches for packages

Package Information

dnf info package-name

Shows package details

List Installed

dnf list installed

Lists all installed packages

Clean Cache

sudo dnf clean all

Cleans package cache

Autoremove

sudo dnf autoremove

Removes unneeded dependencies

Pacman - Package Manager

Arch Manjaro EndeavourOS Garuda

System Update

sudo pacman -Syu

Updates package list and upgrades system

Install Package

sudo pacman -S package-name

Installs a package

Remove Package

sudo pacman -R package-name

Removes a package

Remove with Dependencies

sudo pacman -Rns package-name

Removes package and unused deps

Search Packages

pacman -Ss keyword

Searches remote repositories

Search Installed

pacman -Qs keyword

Searches installed packages

Package Info

pacman -Si package-name

Shows package information

List Installed

pacman -Q

Lists all installed packages

Clean Cache

sudo pacman -Sc

Cleans package cache

⚠️ AUR Helper

For AUR packages, use helpers like yay or paru:

yay -S package-name  # Install from AUR

APK - Alpine Package Keeper

Alpine

Update Repository Index

apk update

Upgrade System

apk upgrade

Install Package

apk add package-name

Remove Package

Search Packages

apk search keyword

Package Info

apk info package-name

Zypper - openSUSE Package Manager

openSUSE SUSE Linux

Refresh Repositories

sudo zypper refresh

Updates repository data

Update System

sudo zypper update

Updates all packages

Install Package

sudo zypper install package-name

Installs a package

Remove Package

sudo zypper remove package-name

Removes a package

Search Packages

zypper search keyword

Searches for packages

Package Info

zypper info package-name

Shows package information

✨ Best Practices

1

Always Update First

Before installing new packages, update your package lists to ensure you get the latest versions.

2

Regular System Updates

Keep your system updated weekly for security patches and bug fixes.

3

Clean Regularly

Remove unused packages and clean cache to free up disk space.

4

Use Official Repositories

Prefer official repos over third-party for security and stability.

5

Read Before Confirming

Always review what will be installed/removed before confirming.

6

Backup Before Major Updates

Create system snapshots before distribution upgrades.
