🧭 A Simple Debug Workflow

1) Identify the scope

Is it one app, one user, one service, or the whole system?

2) Gather evidence

Logs, timestamps, error messages, and recent changes.

3) Isolate variables

Test in safe mode, another user, or a minimal config.

4) Apply minimal fixes

Change one thing at a time, then re-test.

📜 Logs & Diagnostics

Journal (systemd)

journalctl -b
journalctl -p warning..alert -b
journalctl -u ssh -b
journalctl -xe

Kernel & boot

dmesg -T | tail -200
sudo journalctl -k -b
sudo systemctl status

System overview

uname -a
lsb_release -a 2>/dev/null || cat /etc/os-release
uptime
free -h

🧩 Common Problems & Quick Checks

🌐

No network / DNS issues

Verify link, IP, DNS, then routes/firewall.

Commands:
  • ip a / ip r
  • ping -c 3 1.1.1.1
  • ping -c 3 example.com
  • resolvectl status
💾

Disk full / I/O problems

Check free space and which paths are consuming it.

Commands:
  • df -h
  • du -xh / | sort -h | tail
  • journalctl --disk-usage
  • sudo dmesg -T | grep -i error
⚙️

Service won’t start

Inspect status and logs, then validate config and permissions.

Commands:
  • systemctl status myservice
  • journalctl -u myservice -b
  • systemctl cat myservice

📈 Performance Debugging

CPU / Memory

top
htop
ps aux --sort=-%cpu | head
ps aux --sort=-%mem | head

Disk / I/O

iostat -xz 1 5 2>/dev/null
vmstat 1 5
sudo smartctl -a /dev/sda 2>/dev/null

Boot time

systemd-analyze
systemd-analyze blame
systemd-analyze critical-chain

Tip: compare before/after

When you change a setting, re-run the same command set and compare the outputs.

🧯 Recovery Basics

⚠️ Be careful on production

Prefer reversible actions (stop/start services, temporary config changes) before deeper repairs.

Safe shell access

# Local console: try switching TTY
Ctrl+Alt+F3

# Single-user rescue (systemd)
# Add at boot: systemd.unit=rescue.target

Rollback config

# Inspect changes
sudo ls -lt /etc | head

# Keep backups of config files you edit
sudo cp /etc/ssh/sshd_config /etc/ssh/sshd_config.bak

Package repair

# Debian/Ubuntu
sudo apt --fix-broken install

# Arch
sudo pacman -Syu

# Fedora
sudo dnf distro-sync
