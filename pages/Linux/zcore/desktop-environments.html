Desktop Environments & Window Managers

Understand how Linux desktops work: Desktop Environment (DE) vs Window Manager (WM), display managers, and the choice between Wayland and X11.

DE Full desktop suite
WM Window handling only
DM Login/session selector

🧩 Core Concepts

🏙️

Desktop Environment (DE)

A complete UI suite: panels, settings, file manager, notifications, and default apps.

Examples:
  • GNOME
  • KDE Plasma
  • XFCE
  • Cinnamon
  • MATE
🪟

Window Manager (WM)

Controls window placement and behavior. Often paired with separate bars/launchers.

Examples:
  • i3
  • sway (Wayland)
  • Openbox
  • awesome
🔑

Display Manager (DM)

Graphical login that starts the chosen session (GNOME, Plasma, i3, etc.).

Common DMs:
  • GDM
  • SDDM
  • LightDM

🧭 Wayland vs X11

Topic Wayland X11
Security model Stronger isolation for apps by default More permissive; legacy model
Compatibility Great for modern DEs; legacy apps via XWayland Maximum legacy compatibility
Fractional scaling Typically better on modern DEs Varies; sometimes more fiddly
Remote desktop Improving; DE-specific solutions Mature ecosystem

✅ Practical takeaway

If your distro defaults to Wayland (common on modern GNOME), try it first. If you hit driver/app issues, switch the session to X11 from your login screen.

📦 Install & Switch Desktops

Debian/Ubuntu (APT)

sudo apt update
sudo apt install gnome-shell gdm3
# or
sudo apt install kde-standard sddm
# or
sudo apt install xfce4 lightdm

Fedora (DNF)

sudo dnf group list --available
sudo dnf group install "KDE Plasma Workspaces"
# or
sudo dnf group install "XFCE Desktop"

Arch (Pacman)

sudo pacman -Syu
sudo pacman -S plasma sddm
# or
sudo pacman -S gnome gdm
# or
sudo pacman -S xfce4 lightdm lightdm-gtk-greeter

Switch sessions

After install, enable your display manager and pick the session at login.

# Enable a display manager
sudo systemctl enable --now gdm
# or
sudo systemctl enable --now sddm
# or
sudo systemctl enable --now lightdm

🎯 Choosing the Right Desktop

Performance-first

XFCE / MATE are often lighter and great for older hardware or VMs.

Customization-first

KDE Plasma offers deep customization for workflow and appearance.

Simplicity-first

GNOME is clean and consistent; best if you like minimal UI.

Tiling workflows

Try i3/sway if you want keyboard-driven window management.
