📚 Introduction

What is RHEL?

Red Hat Enterprise Linux (RHEL) is a commercial open-source Linux distribution developed by Red Hat for the commercial market. RHEL is based on Fedora but undergoes extensive testing and certification to ensure stability and reliability for enterprise deployments. It's the foundation for many other distributions including CentOS, Rocky Linux, and AlmaLinux.

Key Features

  • Long-term Support: 10 years of security updates and support
  • SELinux: Advanced security framework enabled by default
  • Certifications: Hardware and software vendor certifications
  • Red Hat Insights: Proactive system health monitoring
  • Ansible Integration: Built-in automation capabilities

⚙️ Installation Guide

Step 1: Get RHEL

Download from: Red Hat Customer Portal

Free for developers: Red Hat Developer Program
Choose: Server, Workstation, or specific use case

Step 2: Create Bootable Media

# Linux
sudo dd if=rhel.iso of=/dev/sdX bs=4M status=progress

# Or use Fedora Media Writer

Step 3: Boot & Install

1. Boot from installation media
2. Select "Install Red Hat Enterprise Linux"
3. Choose language and keyboard
4. Configure installation destination
5. Set network and hostname

Step 4: Register System

# Register with Red Hat
sudo subscription-manager register

# Attach subscription
sudo subscription-manager attach --auto

# Enable repositories
sudo subscription-manager repos --enable=rhel-9-for-x86_64-baseos-rpms
⚠️ System Requirements
  • CPU: 2 GHz dual-core or better (64-bit)
  • RAM: 2GB minimum (4GB+ recommended)
  • Storage: 20GB minimum (40GB+ recommended)
  • GPU: For Workstation: graphics card with 1024x768

📖 User Guide

Package Management with DNF

Update System

sudo dnf update -y

Install Package

sudo dnf install package-name

# Install group
sudo dnf group install "Development Tools"

Remove Package

sudo dnf remove package-name
sudo dnf autoremove

Search Package

dnf search keyword
dnf info package-name

Essential Tasks

Subscription Management

# Check status
sudo subscription-manager status

# List available subscriptions
sudo subscription-manager list --available

Enable EPEL Repository

# Install EPEL
sudo dnf install https://dl.fedoraproject.org/pub/epel/epel-release-latest-9.noarch.rpm

Firewall Management

# Check firewall status
sudo firewall-cmd --state

# Add service
sudo firewall-cmd --add-service=http --permanent
sudo firewall-cmd --reload

SELinux Management

# Check SELinux status
sestatus

# View denials
sudo ausearch -m avc -ts recent

🚀 Advanced Topics

Red Hat Insights

# Register with Insights
sudo insights-client --register

# Run analysis
sudo insights-client

# Check status
sudo insights-client --status

Proactive system health monitoring

Container Management (Podman)

# Install Podman
sudo dnf install podman

# Run container
podman run -it registry.access.redhat.com/ubi9/ubi bash

# List containers
podman ps -a

Enterprise container runtime

Ansible Automation

# Install Ansible
sudo dnf install ansible-core

# Run playbook
ansible-playbook playbook.yml

# Ad-hoc command
ansible all -m ping

Automate system configuration

System Roles

# Install system roles
sudo dnf install rhel-system-roles

# Available roles
ls /usr/share/ansible/roles/

# Example: Configure time sync
ansible-playbook -i inventory timesync.yml

Pre-built Ansible roles for common tasks
