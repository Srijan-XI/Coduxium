📚 Introduction

What is Rocky Linux?

Rocky Linux is a community-driven enterprise operating system founded by Gregory Kurtzer, one of the original founders of CentOS. Created in response to Red Hat's decision to discontinue CentOS as a RHEL rebuild, Rocky Linux aims to provide a stable, production-ready platform that is 100% bug-for-bug compatible with RHEL. It's named after Rocky McGaugh, a co-founder of CentOS who passed away in 2004.

Key Features

  • RHEL Compatibility: Drop-in replacement for RHEL and CentOS
  • Long-term Support: 10 years of support matching RHEL lifecycle
  • No Subscription: Free to use with no licensing costs
  • Community Governance: Transparent, community-led development
  • Enterprise Features: SELinux, firewalld, and enterprise tools included

⚙️ Installation Guide

Step 1: Download

Download from: rockylinux.org/download

Choose: Minimal, DVD, or Boot ISO
File size: ~2-10GB depending on version

Step 2: Create Bootable USB

# Linux
sudo dd if=Rocky-9.iso of=/dev/sdX bs=4M status=progress

# Or use Fedora Media Writer, Etcher

Step 3: Boot & Install

1. Boot from installation media
2. Select "Install Rocky Linux"
3. Choose language and keyboard
4. Configure installation destination
5. Set network and hostname
6. Create user account

Step 4: Post-Installation

# Update system
sudo dnf update -y

# Install EPEL (optional)
sudo dnf install epel-release -y

# Reboot
sudo reboot
⚠️ System Requirements
  • CPU: 2 GHz dual-core or better (64-bit)
  • RAM: 2GB minimum (4GB+ recommended)
  • Storage: 20GB minimum (40GB+ recommended)
  • GPU: Not required for server, any for desktop

📖 User Guide

Package Management with DNF

Update System

sudo dnf update -y

Install Package

sudo dnf install package-name

# Install group
sudo dnf group install "Development Tools"

Remove Package

sudo dnf remove package-name
sudo dnf autoremove

Search Package

dnf search keyword
dnf info package-name

Essential Tasks

Install Desktop Environment

# GNOME
sudo dnf group install "Server with GUI"

# Or specific DE
sudo dnf group install "GNOME Desktop"

Firewall Management

# Check status
sudo firewall-cmd --state

# Add service
sudo firewall-cmd --add-service=http --permanent
sudo firewall-cmd --reload

SELinux Management

# Check status
sestatus

# Temporarily set permissive
sudo setenforce 0

Enable EPEL

sudo dnf install epel-release -y
sudo dnf update

🚀 Advanced Topics

Migrate from CentOS

# Download migration script
curl https://raw.githubusercontent.com/rocky-linux/rocky-tools/main/migrate2rocky/migrate2rocky.sh -o migrate2rocky.sh

# Make executable
chmod +x migrate2rocky.sh

# Run migration
sudo ./migrate2rocky.sh -r

Migrate existing CentOS installation to Rocky

Container Management (Podman)

# Install Podman
sudo dnf install podman -y

# Run container
podman run -it rockylinux/rockylinux:9 bash

# List containers
podman ps -a

Container runtime for Rocky Linux

Web Server Setup (Apache)

# Install Apache
sudo dnf install httpd -y

# Start and enable
sudo systemctl start httpd
sudo systemctl enable httpd

# Open firewall
sudo firewall-cmd --add-service=http --permanent
sudo firewall-cmd --reload

Set up web server

Kernel Management

# List installed kernels
rpm -q kernel

# Set default kernel
sudo grubby --set-default /boot/vmlinuz-

# Remove old kernels
sudo dnf remove kernel-

Manage system kernels
