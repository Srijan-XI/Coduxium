📚 Introduction

What is Fedora?

Fedora is a Linux distribution developed by the Fedora Project and sponsored by Red Hat. It serves as the upstream source for Red Hat Enterprise Linux (RHEL), meaning new features and technologies are tested in Fedora before making their way to RHEL. Fedora is known for its commitment to free software, rapid adoption of new technologies, and strong security defaults including SELinux.

Key Features

  • Latest Software: Always includes the newest stable versions of software
  • Wayland by Default: Modern display server protocol for better graphics
  • SELinux: Security-Enhanced Linux enabled and configured out of the box
  • DNF Package Manager: Fast, powerful package management with excellent dependency resolution
  • Multiple Editions: Workstation, Server, IoT, and community spins for different use cases

⚙️ Installation Guide

Step 1: Download

Download Fedora from: getfedora.org

Choose: Fedora Workstation (desktop) or specific spins
File size: ~2-3GB

Step 2: Create Bootable USB

# Linux
sudo dd if=Fedora.iso of=/dev/sdX bs=4M status=progress

# Or use Fedora Media Writer (recommended)
sudo dnf install mediawriter

Step 3: Boot from USB

1. Insert USB drive
2. Restart computer
3. Enter BIOS/UEFI (F2/F12/Del)
4. Select USB as boot device
5. Choose "Start Fedora"

Step 4: Install Fedora

1. Click "Install to Hard Drive"
2. Select language and keyboard
3. Configure disk partitioning
4. Set network and hostname
5. Create user account
6. Begin installation
7. Reboot and enjoy!

⚠️ System Requirements
  • CPU: 2 GHz dual-core or better
  • RAM: 2GB minimum (4GB+ recommended)
  • Storage: 20GB minimum (40GB+ recommended)
  • GPU: Graphics card with OpenGL 2.0+ support

📖 User Guide

Package Management with DNF

Update System

sudo dnf update
# or upgrade (same as update)
sudo dnf upgrade

Install Package

sudo dnf install package-name

# Install group
sudo dnf group install "Development Tools"

Remove Package

sudo dnf remove package-name

# Remove with dependencies
sudo dnf autoremove package-name

Search Package

dnf search keyword
dnf info package-name  # Details

Essential Tasks

GNOME Software Center

Access from Activities → Software to browse and install applications

Enable RPM Fusion

sudo dnf install \
  https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm

Install Media Codecs

sudo dnf install gstreamer1-plugins-{bad-\*,good-\*,base} gstreamer1-plugin-openh264 gstreamer1-libav --exclude=gstreamer1-plugins-bad-free-devel

Flatpak Support

flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

🚀 Advanced Topics

Enable NVIDIA Drivers

# Add RPM Fusion first, then:
sudo dnf install akmod-nvidia
sudo dnf install xorg-x11-drv-nvidia-cuda

Install proprietary NVIDIA graphics drivers

System Rollback with OSTree

# List deployments
rpm-ostree status

# Rollback to previous version
rpm-ostree rollback

# Rebase to different version
rpm-ostree rebase fedora:fedora/39/x86_64/silverblue

Fedora Silverblue/Kinoite feature

Development Tools Setup

# Install development group
sudo dnf group install "Development Tools" "Development Libraries"

# Install languages
sudo dnf install python3-devel nodejs npm golang rust cargo

# Install toolbox for containerized dev
sudo dnf install toolbox
toolbox create fedora-toolbox

Set up complete development environment

SELinux Management

# Check SELinux status
sestatus

# Temporarily set to permissive
sudo setenforce 0

# View SELinux denials
sudo ausearch -m avc -ts recent

# Troubleshoot issues
sudo sealert -a /var/log/audit/audit.log

Manage SELinux security policies

💻 Useful Commands

Task Command Description
Update System sudo dnf update Update all packages to latest version
Install Package sudo dnf install package Install a new package
Remove Package sudo dnf remove package Uninstall a package
Search Packages dnf search keyword Search for available packages
List Installed dnf list installed Show all installed packages
Clean Cache sudo dnf clean all Clear DNF cache
Fedora Version cat /etc/fedora-release Show Fedora version
Service Management sudo systemctl restart service Restart a service
Firewall Control sudo firewall-cmd --list-all View firewall rules
Check Updates dnf check-update List available updates
