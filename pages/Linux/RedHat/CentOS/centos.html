📚 Introduction

What is CentOS Stream?

CentOS Stream is a continuously delivered Linux distribution that provides a rolling preview of what's coming in the next minor release of Red Hat Enterprise Linux. Unlike the old CentOS Linux which was a downstream rebuild of RHEL, CentOS Stream sits between Fedora and RHEL in the development pipeline. This makes it an ideal platform for developers who want to contribute to or test upcoming RHEL features.

Key Features

  • Continuous Delivery: Rolling updates with latest features
  • RHEL Preview: Test upcoming RHEL features before release
  • Development Platform: Contribute to RHEL development
  • Stable Rolling: More stable than Fedora, newer than RHEL
  • Enterprise Tools: SELinux, firewalld, and enterprise features

⚙️ Installation Guide

Step 1: Download

Download from: centos.org/centos-stream

Choose: Boot, DVD, or Minimal ISO
File size: ~2-10GB depending on version

Step 2: Create Bootable USB

# Linux
sudo dd if=CentOS-Stream.iso of=/dev/sdX bs=4M status=progress

# Or use Fedora Media Writer

Step 3: Boot & Install

1. Boot from installation media
2. Select "Install CentOS Stream"
3. Choose language and keyboard
4. Configure installation destination
5. Set network and hostname
6. Create user account

Step 4: Post-Installation

# Update system
sudo dnf update -y

# Install EPEL (optional)
sudo dnf install epel-release -y

# Reboot
sudo reboot
⚠️ System Requirements
  • CPU: 2 GHz dual-core or better (64-bit)
  • RAM: 2GB minimum (4GB+ recommended)
  • Storage: 20GB minimum (40GB+ recommended)
  • GPU: Not required for server

📖 User Guide

Package Management with DNF

Update System

sudo dnf update -y

Install Package

sudo dnf install package-name

# Install group
sudo dnf group install "Development Tools"

Remove Package

sudo dnf remove package-name
sudo dnf autoremove

Search Package

dnf search keyword
dnf info package-name

Essential Tasks

Install Desktop Environment

# GNOME
sudo dnf group install "Server with GUI"

# Start GUI
sudo systemctl set-default graphical.target
sudo reboot

Firewall Management

# Check status
sudo firewall-cmd --state

# Add service
sudo firewall-cmd --add-service=http --permanent
sudo firewall-cmd --reload

SELinux Management

# Check status
sestatus

# View denials
sudo ausearch -m avc -ts recent

Enable EPEL

sudo dnf install epel-release -y
sudo dnf update

🚀 Advanced Topics

Convert to Stream

# Convert CentOS Linux to Stream
sudo dnf install centos-release-stream -y
sudo dnf swap centos-linux-repos centos-stream-repos -y
sudo dnf distro-sync -y

Migrate from CentOS Linux to Stream

Container Management (Podman)

# Install Podman
sudo dnf install podman -y

# Run container
podman run -it quay.io/centos/centos:stream9 bash

# List containers
podman ps -a

Container runtime for CentOS Stream

Web Server Setup (Nginx)

# Install Nginx
sudo dnf install nginx -y

# Start and enable
sudo systemctl start nginx
sudo systemctl enable nginx

# Open firewall
sudo firewall-cmd --add-service=http --permanent
sudo firewall-cmd --reload

Set up web server

Development Tools

# Install development group
sudo dnf group install "Development Tools" -y

# Install languages
sudo dnf install python3-devel nodejs npm golang -y

# Install Git
sudo dnf install git -y

Set up development environment
