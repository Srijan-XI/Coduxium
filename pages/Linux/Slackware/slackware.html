📚 Introduction

What is Slackware?

Slackware is a long-running, independent distribution known for its simplicity and minimal abstraction. It avoids unnecessary distro-specific tooling and expects users to manage configuration thoughtfully.

Key Features

  • Classic UNIX-like philosophy
  • Conservative, stable releases
  • No forced dependency resolution
  • Simple init and configuration patterns
  • Excellent learning platform

Why Choose Slackware?

1

Transparency

Less automation means fewer surprises.

2

Stability

Suitable for systems that value predictability.

3

Learning

Build deep Linux understanding through manual config.

⚙️ Installation Guide

Step 1: Download

Download from: http://www.slackware.com/getslack/

Step 2: Create Install Media

# Linux (USB)
sudo dd if=slackware.iso of=/dev/sdX bs=4M status=progress && sync

Step 3: Boot and Partition

Partition disks carefully. Consider separate /home and swap as needed.

Step 4: Install & Configure

Run installer, select package sets, configure bootloader and network.

⚠️ System Requirements
  • CPU: 64-bit recommended
  • RAM: 2GB minimum (4GB+ recommended)
  • Storage: 20GB+ recommended
  • Skill: Comfortable with manual configuration

📖 User Guide

Package Management

Install Package

installpkg package.tgz

Upgrade Package

upgradepkg package.tgz

Remove Package

removepkg package

Update (slackpkg)

slackpkg update
slackpkg upgrade-all

Essential Tasks

Read Docs

Slackware rewards reading official docs and Slackbook.

Keep Backups

Manual systems benefit from well-documented backups.

Track Changes

Keep a changelog for manual config edits.

Secure Services

Only enable services you actively use.

🚀 Advanced Topics

Runlevels / init scripts

ls /etc/rc.d/
cat /etc/inittab

Understand init and startup scripts.

Kernel Info

uname -a
ls /boot

Inspect kernel and boot artifacts.

Network Debug

ip a
route -n

Basic networking visibility.

System Logs

dmesg | tail
tail -n 100 /var/log/messages

Debug hardware and service issues.
