Distribution Families

Linux distributions are categorized into families based on their package management and lineage. Choose the one that fits your needs.

🏹

Arch Family

Pacman • Rolling

Philosophy: Simplicity, DIY, Power-user centric.

  • Arch Linux Base
  • BlackArch Security
  • Garuda Performance
  • RedArch Tools
🌀

Debian Family

APT • Stable

Philosophy: Stability, Free Software, User-friendly derivatives.

  • Ubuntu Popular
  • Kali Linux Security
  • Linux Mint Beginner
  • ParrotSec Privacy
🎩

Red Hat Family

DNF/RPM • Enterprise

Philosophy: Corporate standards, Innovation, Stability.

  • Fedora Innovation
  • RHEL Enterprise
  • CentOS Server
  • Oracle Stable
💜

Gentoo Family

Portage • Source

Philosophy: Optimization, Choice, Compile-everything.

  • Gentoo Configurable
  • USE Flags Granular
⚙️

Slackware Family

pkgtools • UNIX-like

Philosophy: Stability, Simplicity, No dependency resolution.

  • Slackware Oldest
  • Manual Hands-on
