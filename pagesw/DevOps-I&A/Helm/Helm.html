📦 Helm

Helm is the package manager for Kubernetes that standardizes app deployment through reusable charts and managed releases.

Overview

Kubernetes Package Manager Charts Releases

Helm simplifies Kubernetes YAML management by turning infrastructure templates and configuration into versioned, reusable chart packages.

Core Concepts

  • Chart — package describing Kubernetes resources
  • Values — environment-specific configuration overrides
  • Release — deployed chart instance in a cluster
  • Repository — chart distribution source

Getting Started

# Install helm and verify
helm version

# Add chart repo and install app
helm repo add bitnami https://charts.bitnami.com/bitnami
helm repo update
helm install my-nginx bitnami/nginx
# Upgrade and rollback
helm upgrade my-nginx bitnami/nginx --set service.type=LoadBalancer
helm rollback my-nginx 1

Best Practices

  • Keep values files per environment for predictable promotion.
  • Pin chart versions in production workflows.
  • Validate templates in CI with helm lint and helm template.
  • Use secrets integrations rather than committing plaintext values.
