🧠 Pulumi

Pulumi enables infrastructure-as-code using familiar languages like TypeScript, Python, Go, C#, and Java.

Overview

IaC Code-first Multi-Cloud Policy as Code

Pulumi combines software engineering patterns with infrastructure workflows, allowing stronger abstraction, testing, and reuse.

Core Concepts

  • Stack — environment-specific infrastructure instance
  • Program — language-native resource definitions
  • Preview — planned change visualization
  • Providers — cloud/service integrations

Getting Started

# Create a new project
pulumi new aws-typescript

# Preview and deploy
pulumi preview
pulumi up
# Destroy stack resources
pulumi destroy
pulumi stack rm

Common Use Cases

  • Application + infrastructure definitions in one codebase
  • Reusable platform abstractions and components
  • Policy-enforced multi-cloud provisioning
  • IaC workflows integrated with standard SDLC tooling
