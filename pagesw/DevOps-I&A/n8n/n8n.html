🔗 n8n

n8n is a workflow automation platform for connecting services and building event-driven automation with low-code and code nodes.

Overview

AutomationIntegrationsWorkflowsLow-code

n8n enables teams to automate operational tasks, data syncs, and business processes with visual workflows and extensible execution logic.

Common Use Cases

  • Cross-SaaS data synchronization
  • Alerting and incident routing automation
  • Ticketing, CRM, and messaging orchestration
  • Internal tool process automation

Getting Started

# Run n8n in Docker
docker run -it --rm -p 5678:5678 n8nio/n8n

# Open UI
http://localhost:5678
