🤗 Hugging Face Transformers

State-of-the-art Natural Language Processing library with thousands of pre-trained models.

Overview

NLP Transformers Python Pre-trained Models

Hugging Face Transformers provides thousands of pre-trained models for Natural Language Processing, Computer Vision, and Audio tasks. It's the go-to library for working with transformer architectures like BERT, GPT, T5, and more.

Key Features

  • Model Hub — 100,000+ pre-trained models
  • Multi-Framework Support — PyTorch, TensorFlow, JAX
  • Pipelines — Simple interface for common tasks
  • AutoModels — Automatic model selection
  • Tokenizers — Fast tokenization library
  • Datasets — Integrated dataset library
  • Fine-Tuning — Easy transfer learning
  • ONNX Export — Production deployment

Common Use Cases

  • Text classification and sentiment analysis
  • Question answering systems
  • Text generation (GPT models)
  • Named entity recognition (NER)
  • Machine translation
  • Text summarization
  • Image classification and object detection
  • Speech recognition and generation

Popular Models

  • BERT — Bidirectional encoder representations
  • GPT-2/GPT-3 — Generative pre-trained transformers
  • T5 — Text-to-text transfer transformer
  • RoBERTa — Robustly optimized BERT
  • BART — Denoising autoencoder
  • DistilBERT — Lightweight BERT variant
  • LLaMA/Mistral — Open-source LLMs
  • CLIP/ViT — Vision transformers

Getting Started

Install Transformers:

pip install transformers
pip install datasets  # Optional but recommended

Using pipelines for quick inference:

from transformers import pipeline

# Sentiment analysis
classifier = pipeline("sentiment-analysis")
result = classifier("I love Hugging Face!")

# Text generation
generator = pipeline("text-generation", model="gpt2")
result = generator("Hello, I'm a language model")

# Question answering
qa = pipeline("question-answering")
result = qa(
    question="What is NLP?",
    context="NLP is Natural Language Processing"
)

Fine-tuning a model:

from transformers import AutoTokenizer, AutoModelForSequenceClassification
from transformers import Trainer, TrainingArguments

# Load model and tokenizer
tokenizer = AutoTokenizer.from_pretrained("bert-base-uncased")
model = AutoModelForSequenceClassification.from_pretrained("bert-base-uncased")

# Prepare training arguments and trainer
training_args = TrainingArguments(output_dir="./results")
trainer = Trainer(model=model, args=training_args, train_dataset=train_dataset)

# Fine-tune
trainer.train()

Best Practices

  • Fine-tune on domain-specific data
