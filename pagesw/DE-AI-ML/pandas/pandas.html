🐼 Pandas

Powerful, flexible data analysis and manipulation library built on top of NumPy and Python.

Overview

Data Analysis Python DataFrames Time Series

Pandas is an open-source library providing high-performance, easy-to-use data structures and data analysis tools for Python. It's the de facto standard for data manipulation and analysis in Python.

Key Features

  • DataFrame — Powerful tabular data structure
  • Data Cleaning — Handle missing data easily
  • Data Manipulation — Merge, join, group, reshape
  • Time Series — Comprehensive time series functionality
  • IO Tools — Read/write CSV, Excel, SQL, JSON, etc.
  • Indexing — Flexible and intuitive indexing
  • Aggregation — GroupBy operations
  • Visualization — Built-in plotting capabilities

Common Use Cases

  • Data cleaning and preprocessing
  • Exploratory data analysis (EDA)
  • Time series analysis
  • Statistical analysis
  • Data transformation and wrangling
  • Financial analysis
  • Scientific computing
  • Business intelligence

Getting Started

Install Pandas:

pip install pandas

Basic operations:

import pandas as pd
import numpy as np

# Create DataFrame
df = pd.DataFrame({
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['NYC', 'LA', 'Chicago']
})

# Read from CSV
df = pd.read_csv('data.csv')

# Basic operations
df.head()                # First 5 rows
df.describe()            # Statistical summary
df['Age'].mean()         # Column mean
df[df['Age'] > 28]      # Filtering

# GroupBy
df.groupBy('City').agg({'Age': 'mean'})

# Save to file
df.to_csv('output.csv', index=False)

Best Practices

  • Use vectorized operations instead of loops
  • Choose appropriate data types to save memory
  • Use categorical data type for repetitive strings
  • Chain methods for cleaner code
  • Handle missing data explicitly
  • Use query() for complex filtering
  • Avoid modifying DataFrames in-place when possible
  • Profile memory usage for large datasets
