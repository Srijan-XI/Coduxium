🔢 NumPy

The fundamental package for scientific computing with Python, providing multi-dimensional arrays and mathematical functions.

Overview

Scientific Computing Python Arrays Mathematics

NumPy is the fundamental package for numerical computing in Python. It provides a powerful N-dimensional array object, sophisticated broadcasting functions, and tools for integrating C/C++ and Fortran code.

Key Features

  • ndarray — Powerful N-dimensional array object
  • Broadcasting — Vectorized operations on arrays
  • Linear Algebra — Comprehensive linear algebra functions
  • Random Number Generation — Various distributions
  • Fourier Transforms — Fast Fourier transform support
  • Performance — Optimized C and Fortran code
  • Integration — Seamless Pandas, SciPy, Matplotlib integration
  • Memory Efficient — Typed arrays for performance

Common Use Cases

  • Scientific and numerical computing
  • Data analysis and statistics
  • Machine learning preprocessing
  • Image processing
  • Signal processing
  • Linear algebra operations
  • Financial calculations
  • Simulation and modeling

Getting Started

Install NumPy:

pip install numpy

Basic operations:

import numpy as np

# Create arrays
arr = np.array([1, 2, 3, 4, 5])
matrix = np.array([[1, 2], [3, 4]])

# Array creation functions
np.zeros((3, 3))          # 3x3 array of zeros
np.ones((2, 4))           # 2x4 array of ones
np.arange(0, 10, 2)       # [0, 2, 4, 6, 8]
np.linspace(0, 1, 5)      # 5 evenly spaced values

# Operations
arr * 2                    # Element-wise multiplication
arr + 10                   # Element-wise addition
np.dot(matrix, matrix)     # Matrix multiplication

# Statistical functions
arr.mean()
arr.std()
arr.max()

# Indexing and slicing
arr[0]                     # First element
arr[1:4]                   # Slice
matrix[0, 1]               # Element at row 0, col 1

# Reshaping
arr.reshape((5, 1))        # Reshape to column vector

Best Practices

  • Use vectorized operations instead of loops
  • Leverage broadcasting for efficient computations
  • Choose appropriate data types for memory efficiency
  • Use views instead of copies when possible
  • Profile code to identify bottlenecks
  • Use np.newaxis for dimension manipulation
  • Understand memory layout (C-order vs F-order)
  • Use einsum for complex array operations
