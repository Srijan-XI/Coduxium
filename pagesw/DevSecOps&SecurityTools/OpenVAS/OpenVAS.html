🔍 OpenVAS

OpenVAS is an open-source vulnerability scanning platform used to identify security weaknesses and prioritize remediation across infrastructure.

Overview

Vulnerability Scanner Risk Assessment Network Security DevSecOps

OpenVAS includes an extensive vulnerability test feed and reporting capabilities, enabling regular security baselines and patch verification workflows.

Key Capabilities

  • Large vulnerability test catalog
  • Configurable scan policies and schedules
  • Asset-based risk reporting
  • Remediation-oriented findings output
  • Ongoing security posture tracking

Getting Started

# Example with Greenbone Community Edition (containerized)
docker run -d --name openvas --publish 9392:9392 immauss/openvas

# Access web UI
# https://localhost:9392

# Workflow
# 1) Create target
# 2) Create task and choose scan config
# 3) Start scan
# 4) Review report and prioritize fixes

Operational Best Practices

  • Run authenticated scans when possible for deeper findings
  • Schedule periodic scans and compare trends
  • Validate critical findings to reduce false positives
  • Track remediation SLA by severity
  • Limit scans to approved assets and change windows
