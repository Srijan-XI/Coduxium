🐍 Scapy

Packet Manipulation Python Network Security

Scapy is a powerful Python-based interactive packet manipulation program and library. It can forge or decode packets, send them on the wire, capture them, and match requests and replies.

Key Features

  • Create, send, and capture network packets
  • Protocol fuzzing and network discovery
  • Packet sniffing and analysis
  • Custom protocol implementation

Basic Usage

from scapy.all import *

# Send ICMP ping
send(IP(dst="8.8.8.8")/ICMP())

# Sniff packets
packets = sniff(count=10)

# ARP scan
ans, unans = srp(Ether(dst="ff:ff:ff:ff:ff:ff")/ARP(pdst="192.168.1.0/24"),
                  timeout=2)

🔍 OpenVAS

Vulnerability Scanner Open Source Network Security

OpenVAS (Open Vulnerability Assessment System) is a full-featured vulnerability scanner with thousands of vulnerability tests to identify security issues in networks and systems.

Key Features

  • 70,000+ vulnerability tests (NVTs)
  • Comprehensive scan configurations
  • Detailed reporting and remediation advice
  • Continuous vulnerability management

Common Use Cases

  • Regular vulnerability assessments
  • Compliance scanning (PCI-DSS, HIPAA)
  • Network security auditing
  • Patch management verification

🔥 Burp Suite

Web Security Proxy Penetration Testing

Burp Suite is an integrated platform for performing security testing of web applications. It's the industry standard for web app penetration testing.

Key Features

  • Proxy — Intercept and modify HTTP/HTTPS traffic
  • Scanner — Automated vulnerability detection
  • Intruder — Automated customized attacks
  • Repeater — Modify and resend requests
  • Sequencer — Test randomness of session tokens
  • Decoder — Encode/decode data

Common Attacks

  • SQL Injection testing
  • XSS (Cross-Site Scripting)
  • CSRF (Cross-Site Request Forgery)
  • Authentication bypass

⚡ OWASP ZAP

Web App Security OWASP Free & Open Source

OWASP Zed Attack Proxy (ZAP) is a free, open-source penetration testing tool for finding vulnerabilities in web applications.

Key Features

  • Automated scanner for common vulnerabilities
  • Intercepting proxy for manual testing
  • Active and passive scanning modes
  • REST API for CI/CD integration
  • Extensive marketplace of add-ons

Integration

# Run ZAP in daemon mode
zap.sh -daemon -port 8080 -config api.key=your-api-key

# Automated scan via API
zap-cli quick-scan http://target-site.com

🎯 YARA

Malware Detection Pattern Matching Threat Hunting

YARA is a tool for identifying and classifying malware samples based on textual or binary patterns. It's the pattern matching Swiss Army knife for security researchers.

Key Features

  • Pattern-based malware identification
  • Custom rule creation
  • Fast file scanning
  • Integration with security tools

Example YARA Rule

rule SuspiciousPowerShell
{
    meta:
        description = "Detects suspicious PowerShell"
        author = "Security Team"
    
    strings:
        $s1 = "Invoke-Expression" nocase
        $s2 = "DownloadString" nocase
        $s3 = "hidden" nocase
    
    condition:
        2 of ($s*)
}

🔧 Radare2

Reverse Engineering Binary Analysis Disassembler

Radare2 is a portable reversing framework for binary analysis, disassembly, debugging, and exploiting. It's a complete toolkit for reverse engineers.

Key Features

  • Multi-architecture disassembler
  • Debugger with GDB remote support
  • Binary diffing and patching
  • Scripting with r2pipe
  • Graph visualization

Basic Commands

# Analyze binary
r2 -A binary

# Show functions
afl

# Disassemble function
pdf @ main

# Set breakpoint
db 0x08048400

# Continue execution
dc
