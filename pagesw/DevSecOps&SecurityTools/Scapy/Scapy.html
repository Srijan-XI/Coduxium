🐍 Scapy

Scapy is a Python packet manipulation toolkit used for crafting, sniffing, and analyzing network traffic in security and diagnostics workflows.

Overview

Network Security Packet Analysis Python Pentesting

Scapy gives fine-grained control over network packets and protocols, making it useful for protocol testing, custom packet generation, and network behavior analysis.

Key Capabilities

  • Craft and send custom packets
  • Sniff and filter live traffic
  • Discover hosts with ARP/ICMP probing
  • Build protocol fuzzing and test scripts
  • Automate repeatable network checks

Getting Started

pip install scapy
from scapy.all import IP, ICMP, send, sniff, Ether, ARP, srp

# Ping test
send(IP(dst="8.8.8.8") / ICMP())

# Capture 5 packets
pkts = sniff(count=5)
print(pkts.summary())

# ARP scan
ans, _ = srp(
    Ether(dst="ff:ff:ff:ff:ff:ff") / ARP(pdst="192.168.1.0/24"),
    timeout=2,
    verbose=False,
)
for _, rcv in ans:
    print(rcv.psrc, rcv.hwsrc)

Best Practices

  • Run only in authorized environments and scopes
  • Use isolated lab networks for protocol experimentation
  • Throttle scans to avoid network disruption
  • Log findings and timestamps for reproducibility
