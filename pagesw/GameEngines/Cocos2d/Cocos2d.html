🎮 Cocos2d

Family of open-source frameworks for building 2D games and interactive applications.

Overview

2D MOBILE CROSS-PLATFORM OPEN SOURCE

Cocos2d is a family of open-source software frameworks for building cross-platform games and apps. The most popular variants are Cocos2d-x (C++), Cocos Creator (JavaScript/TypeScript), and the original Cocos2d (Python).

Cocos2d Variants

  • Cocos2d-x — C++ framework for mobile and desktop games
  • Cocos Creator — Visual editor with JavaScript/TypeScript scripting
  • Cocos2d — Original Python framework for iOS
  • Cocos2d-js — JavaScript framework (merged into Creator)

Key Features

  • Cross-Platform — iOS, Android, Windows, macOS, Linux, Web
  • Performance — Optimized for mobile devices
  • Visual Editor — Cocos Creator provides intuitive UI
  • Animation — Timeline animation system
  • Physics — Integrated physics engines
  • Rich UI — Complete UI widget system
  • Particle System — Built-in particle effects
  • Audio — Multi-channel audio support

Cocos Creator Installation

# Download Cocos Creator from official website
# https://www.cocos.com/en/creator/download

# Or install via npm for Cocos2d-js
npm install -g cocos2d-js

# Create new project
cocos new MyGame -l js

Simple Scene Example (Cocos Creator)

// TypeScript component example
const { ccclass, property } = cc._decorator;

@ccclass
export default class Player extends cc.Component {
    @property(cc.Float)
    speed: number = 200;

    onLoad() {
        cc.systemEvent.on(cc.SystemEvent.EventType.KEY_DOWN, 
            this.onKeyDown, this);
    }

    onKeyDown(event) {
        switch(event.keyCode) {
            case cc.macro.KEY.left:
                this.node.x -= this.speed;
                break;
            case cc.macro.KEY.right:
                this.node.x += this.speed;
                break;
        }
    }
}
