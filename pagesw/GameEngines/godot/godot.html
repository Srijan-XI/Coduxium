🎯 Godot Engine

Free and open source game engine for creating 2D and 3D games with an innovative scene system.

Overview

Game Engine Open Source 2D/3D GDScript

Godot is a feature-packed, cross-platform game engine completely free and open source under the MIT license. It's designed to create both 2D and 3D games with an intuitive scene system and powerful scripting.

Key Features

  • 100% Free — No royalties, no subscriptions, truly yours
  • Scene System — Innovative node and scene architecture
  • GDScript — Python-like scripting language
  • Visual Scripting — Node-based programming
  • Excellent 2D — Dedicated 2D engine with pixel-perfect rendering
  • Modern 3D — PBR rendering, global illumination
  • Cross-Platform — Export to desktop, mobile, web
  • Live Editing — Edit scenes while game is running

Common Use Cases

  • 2D games (platformers, RPGs, puzzle games)
  • 3D games (FPS, adventure, simulation)
  • Mobile game development
  • Web games
  • Educational games and simulations
  • Indie game development
  • Prototyping and game jams
  • Interactive applications

Getting Started

GDScript example:

# Player.gd
extends CharacterBody2D

var speed = 200.0
var jump_velocity = -400.0

func _physics_process(delta):
    # Add gravity
    if not is_on_floor():
        velocity.y += get_gravity().y * delta
    
    # Handle jump
    if Input.is_action_just_pressed("ui_accept") and is_on_floor():
        velocity.y = jump_velocity
    
    # Get input direction
    var direction = Input.get_axis("ui_left", "ui_right")
    if direction:
        velocity.x = direction * speed
    else:
        velocity.x = move_toward(velocity.x, 0, speed)
    
    move_and_slide()

# Signal example
signal health_changed(new_health)

func take_damage(amount):
    health -= amount
    emit_signal("health_changed", health)

  • Profile performance regularly
