☕ LibGDX

LibGDX is a Java game development framework for building cross-platform 2D and 3D games from a shared core.

Overview

Game FrameworkJavaCross-platform2D/3D

LibGDX targets desktop, Android, iOS, and web builds with a single gameplay codebase plus platform launchers.

Key Features

  • Shared core module architecture
  • Graphics, audio, and input APIs
  • Scene2D UI toolkit
  • Box2D and asset management support

Getting Started

# Use gdx-liftoff to create a project
# Then run desktop target with Gradle
./gradlew desktop:run

Common Use Cases

  • Mobile and desktop indie games
  • Arcade and puzzle mechanics
  • Java-based classroom game projects
