❤️ Love2D

Love2D is a lightweight Lua framework for building 2D games with fast iteration and simple project structure.

Overview

Game FrameworkLua2DIndie

Love2D uses a callback-style architecture that makes it ideal for prototypes, game jams, and small-to-medium 2D productions.

Key Features

  • Minimal setup and API surface
  • Rapid prototyping loops
  • Simple rendering and input handling
  • Strong community package ecosystem

Getting Started

-- main.lua
function love.update(dt)
end

function love.draw()
  love.graphics.print("Hello Love2D", 100, 100)
end

Common Use Cases

  • 2D indie games
  • Game jam prototypes
  • Learning game loop fundamentals
