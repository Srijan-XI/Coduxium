🎮 GameMaker

Accessible 2D game development engine perfect for beginners and indie developers.

Overview

2D GML INDIE VISUAL SCRIPTING

GameMaker (formerly GameMaker Studio) is a cross-platform game engine developed by YoYo Games. Known for its intuitive drag-and-drop interface and powerful GML (GameMaker Language) scripting, it's the engine behind many successful indie games.

Key Features

  • Drag-and-Drop — Visual programming for beginners
  • GML Scripting — C-like language for advanced logic
  • Cross-Platform — Windows, macOS, Linux, HTML5, iOS, Android, consoles
  • Built-in Editor — Integrated sprite, room, and animation editors
  • Tile System — Efficient tile-based level design
  • Physics — Box2D physics integration
  • Particles — Flexible particle system
  • Audio — Complete audio management

Getting Started

# Download GameMaker
# https://gamemaker.io/en/download

# Installation
1. Download installer for your platform
2. Run installer
3. Create YoYo Games account
4. Choose license tier

# Create First Project
1. File → New Project
2. Choose DnD or GML
3. Add sprites, objects, rooms
4. Run with F5

GML Code Example

// Create Event - Initialize player
speed = 0;
max_speed = 5;
acceleration = 0.5;
friction_amount = 0.3;

// Step Event - Movement logic
if (keyboard_check(vk_right)) {
    speed = min(speed + acceleration, max_speed);
    sprite_index = spr_player_run;
    image_xscale = 1;
}
else if (keyboard_check(vk_left)) {
    speed = max(speed - acceleration, -max_speed);
    sprite_index = spr_player_run;
    image_xscale = -1;
}
else {
    // Apply friction
    if (speed > 0) {
        speed = max(speed - friction_amount, 0);
    } else if (speed < 0) {
        speed = min(speed + friction_amount, 0);
    }
    sprite_index = spr_player_idle;
}

// Move the player
x += speed;

// Jump logic
if (keyboard_check_pressed(vk_space) && place_meeting(x, y+1, obj_ground)) {
    vspeed = -10;
}

// Gravity
if (!place_meeting(x, y+1, obj_ground)) {
    vspeed += 0.5;
}
