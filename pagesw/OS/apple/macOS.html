🍎 macOS

Apple's Unix-based operating system combining elegant design with powerful Unix underpinnings.

Overview

UNIX DESIGN PREMIUM CREATIVE

macOS (formerly Mac OS X and OS X) is Apple's proprietary Unix-based operating system for Mac computers. Built on a Darwin foundation (BSD Unix), it combines the power of Unix with Apple's renowned design philosophy and ecosystem integration.

Evolution & Versions

  • Mac OS X 10.0 "Cheetah" (2001) — First OS X release
  • Mac OS X 10.4 "Tiger" (2005) — Spotlight search, Dashboard widgets
  • Mac OS X 10.5 "Leopard" (2007) — Time Machine, 64-bit apps
  • Mac OS X 10.6 "Snow Leopard" (2009) — Refinement and optimization
  • Mac OS X 10.7 "Lion" (2011) — iOS-inspired features, Mission Control
  • OS X 10.8 "Mountain Lion" (2012) — Notification Center, iCloud
  • OS X 10.10 "Yosemite" (2014) — Flat design, Handoff
  • macOS 10.12 "Sierra" (2016) — Siri on Mac, Universal Clipboard
  • macOS 10.13 "High Sierra" (2017) — APFS file system
  • macOS 10.14 "Mojave" (2018) — Dark Mode, Desktop Stacks
  • macOS 10.15 "Catalina" (2019) — Music, TV, Podcasts apps; Catalyst
  • macOS 11 "Big Sur" (2020) — UI redesign, Apple Silicon support
  • macOS 12 "Monterey" (2021) — Universal Control, Shortcuts
  • macOS 13 "Ventura" (2022) — Stage Manager, Continuity Camera
  • macOS 14 "Sonoma" (2023) — Widgets on Desktop, Game Mode
  • macOS 15 "Sequoia" (2024) — iPhone Mirroring, Apple Intelligence

Key Features

  • Unix Foundation: Darwin (BSD Unix) kernel with full POSIX compliance
  • Aqua Interface: Elegant, consistent user interface design
  • Spotlight: System-wide instant search
  • Time Machine: Automatic backup solution
  • Mission Control: Window and desktop management
  • Continuity: Seamless integration with iPhone, iPad, Apple Watch
  • Handoff: Start on one device, finish on another
  • AirDrop: Wireless file sharing between Apple devices
  • iCloud: Cloud storage and synchronization
  • FileVault: Full-disk encryption
  • Gatekeeper: App security and code signing

Apple Silicon (M1/M2/M3/M4)

# Check architecture
uname -m
# arm64 = Apple Silicon, x86_64 = Intel

# Rosetta 2 (for Intel apps on Apple Silicon)
softwareupdate --install-rosetta --agree-to-license

# Run command with Rosetta
arch -x86_64 /bin/bash

# Universal Binary check
file /Applications/Safari.app/Contents/MacOS/Safari

# Install Homebrew for Apple Silicon
# ARM64 (native)
/opt/homebrew/bin/brew install package

# x86_64 (Rosetta)
arch -x86_64 /usr/local/bin/brew install package

Keyboard Shortcuts

  • ⌘ + Space: Spotlight search
  • ⌘ + Tab: Switch between applications
  • ⌘ + `: Switch between windows of same app
  • ⌘ + Q: Quit application
  • ⌘ + W: Close window
  • ⌘ + ,: Open preferences
  • ⌃ + ↑: Mission Control
  • ⌃ + ←/→: Switch between desktops
  • ⌘ + Shift + 3: Screenshot entire screen
  • ⌘ + Shift + 4: Screenshot selection
  • ⌘ + Shift + 5: Screenshot/Screen recording toolbar

Automation with AppleScript

-- Display notification
display notification "Build completed" with title "Xcode"

-- Dialog box
display dialog "Are you sure?" buttons {"Cancel", "OK"} default button "OK"

-- Open application
tell application "Safari"
    activate
    open location "https://example.com"
end tell

-- System Events
tell application "System Events"
    -- Get list of processes
    set processList to name of every process
end tell

-- Folder actions
on adding folder items to thisFolder after receiving addedItems
    repeat with anItem in addedItems
        -- Process new file
        display notification "New file added: " & (name of anItem)
    end repeat
end adding folder items to

# Run AppleScript from terminal
osascript -e 'display notification "Hello" with title "Test"'
osascript script.scpt

Python Scripting on macOS

import subprocess
import os

# macOS specific - Open file/URL
os.system('open https://apple.com')
os.system('open -a Safari')

# Run AppleScript from Python
script = '''
tell application "Finder"
    activate
    make new Finder window
end tell
'''
subprocess.run(['osascript', '-e', script])

# System notifications
subprocess.run([
    'osascript', '-e',
    'display notification "Build complete" with title "Python Script"'
])

Security Features

  • System Integrity Protection (SIP): Protects system files from modification
  • Gatekeeper: Verifies apps from identified developers
  • XProtect: Built-in malware detection
  • FileVault: Full-disk encryption (AES-XTS 256-bit)
  • Touch ID/Face ID: Biometric authentication
  • Secure Enclave: Hardware security for encryption keys
  • App Sandbox: Isolates apps from system resources
  • Notarization: Apple verification for distributed apps

Continuity Features

  • Universal Clipboard: Copy on one device, paste on another
  • Handoff: Continue work across devices
  • Universal Control: Use one keyboard/mouse across Mac and iPad
  • Sidecar: Use iPad as second display
  • AirDrop: Wireless file transfer
  • iPhone Mirroring: Control iPhone from Mac (macOS Sequoia)
  • Continuity Camera: Use iPhone as webcam
