🪟 Microsoft Windows

The world's most widely used desktop operating system, powering billions of devices worldwide.

Overview

DESKTOP GUI ENTERPRISE GAMING

Microsoft Windows is a family of proprietary graphical operating systems developed by Microsoft. Since its introduction in 1985, Windows has evolved through multiple generations, becoming the dominant OS for personal computers, enterprise environments, and gaming.

Windows 95 (1995)

LEGACY START MENU
  • Released: August 24, 1995
  • Revolutionary Features: Introduced Start Menu, Taskbar, and Desktop icons
  • 32-bit Architecture: Hybrid 16/32-bit system
  • Plug and Play: Automatic hardware detection
  • Internet Explorer: Built-in web browser (IE 1.0, later updated to IE 4.0)
  • Long Filenames: Support for filenames up to 255 characters
  • Legacy: Defined the modern Windows UI paradigm

Windows 98 (1998)

LEGACY USB SUPPORT
  • Released: June 25, 1998
  • USB Support: First Windows with native USB support
  • FAT32: Support for larger hard drives (>2GB)
  • Windows Update: Introduced automatic updates
  • IE 4.0/5.0: Integrated Internet Explorer with Active Desktop
  • Second Edition (SE): Released in 1999 with IE 5.0 and better hardware support
  • Impact: Most popular OS of the late 90s

Windows XP (2001)

ICONIC STABLE
  • Released: October 25, 2001
  • NT Kernel: Built on Windows NT 5.1 (rock-solid stability)
  • Luna Theme: Colorful Fisher-Price interface (Blue, Olive, Silver)
  • Editions: Home, Professional, Media Center, Tablet PC
  • Fast User Switching: Multiple user accounts without logging off
  • System Restore: Rollback to previous states
  • Remote Desktop: Built-in RDP support
  • Service Pack 2 (2004): Major security overhaul with Windows Firewall
  • Longevity: Supported until 2014 (13 years!) - most beloved Windows version

Windows Vista (2007)

CONTROVERSIAL AERO
  • Released: January 30, 2007
  • Windows Aero: Glass transparency effects and Desktop Window Manager
  • User Account Control (UAC): Security feature (heavily criticized)
  • Windows Defender: Built-in antispyware
  • Windows Search: Desktop search indexing
  • SuperFetch: Intelligent memory management
  • DirectX 10: Enhanced gaming graphics
  • Problems: High hardware requirements, driver compatibility issues, performance problems
  • Reception: Poor - many users stayed on XP or downgraded

Windows 7 (2009)

REDEMPTION STABLE
  • Released: October 22, 2009
  • Vista Done Right: Refined and optimized Vista's features
  • Taskbar Overhaul: Jump Lists, pinned apps, live previews
  • Performance: Faster boot times, reduced memory footprint
  • Libraries: Virtual folders for organizing files
  • HomeGroup: Simplified home networking
  • Touch Support: Multi-touch gestures
  • DirectX 11: Advanced graphics API
  • Success: Widely adopted, second most beloved after XP
  • Support Ended: January 14, 2020

Windows 10 (2015)

MODERN UNIFIED
  • Released: July 29, 2015
  • Free Upgrade: Free for Windows 7/8.1 users (first year)
  • Start Menu Hybrid: Combined Classic Start with Live Tiles
  • Cortana: Integrated digital assistant
  • Microsoft Edge: New browser replacing Internet Explorer
  • Virtual Desktops: Multiple workspace support
  • DirectX 12: Gaming performance improvements
  • Windows Subsystem for Linux (WSL): Run Linux binaries natively
  • Continuum: Adaptive UI for tablet/desktop modes
  • Windows Hello: Biometric authentication (face, fingerprint)
  • Rolling Updates: Feature updates twice yearly (21H1, 21H2, etc.)
  • Gaming: Game Bar, Game Mode, DirectStorage

Windows 11 (2021)

LATEST CENTERED
  • Released: October 5, 2021
  • Centered Taskbar: macOS-inspired centered Start and icons
  • Rounded Corners: Modern, rounded UI design language
  • Widgets: Glanceable information panel
  • Snap Layouts: Enhanced window management with templates
  • Android Apps: Run Android apps via Amazon Appstore (WSA)
  • Teams Integration: Microsoft Teams built into taskbar
  • DirectStorage: Faster game loading from NVMe SSDs
  • Auto HDR: Automatic HDR for games
  • TPM 2.0 Requirement: Controversial hardware requirement
  • Performance: Better battery life, improved multitasking
  • Annual Updates: One major update per year (22H2, 23H2, 24H2)

Windows PowerShell Examples

# System Information
Get-ComputerInfo
systeminfo

# Windows Version
winver
[System.Environment]::OSVersion

# List installed programs
Get-WmiObject -Class Win32_Product | Select-Object Name, Version

# Check Windows Update
Get-WindowsUpdate
Install-WindowsUpdate

# Disk usage
Get-PSDrive
Get-Volume

# Running processes
Get-Process | Sort-Object CPU -Descending | Select-Object -First 10

# Network configuration
Get-NetIPConfiguration
ipconfig /all

# Windows Features
Get-WindowsOptionalFeature -Online
Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

# Task Scheduler
Get-ScheduledTask
Register-ScheduledTask -TaskName "MyTask" -Trigger (New-ScheduledTaskTrigger -Daily -At 9am) -Action (New-ScheduledTaskAction -Execute "notepad.exe")

# File operations
Get-ChildItem -Path C:\ -Recurse -Filter *.log
Copy-Item -Path "source.txt" -Destination "dest.txt"
Remove-Item -Path "file.txt" -Force

Windows Registry Tweaks

# Enable Dark Mode (Windows 10/11)
Set-ItemProperty -Path HKCU:\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize -Name AppsUseLightTheme -Value 0

# Disable Windows Telemetry
Set-ItemProperty -Path HKLM:\SOFTWARE\Policies\Microsoft\Windows\DataCollection -Name AllowTelemetry -Value 0

# Disable Cortana
Set-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows\Windows Search" -Name AllowCortana -Value 0

# Show file extensions
Set-ItemProperty -Path HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced -Name HideFileExt -Value 0

# Show hidden files
Set-ItemProperty -Path HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced -Name Hidden -Value 1

# Disable startup delay
Set-ItemProperty -Path HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\Serialize -Name StartupDelayInMSec -Value 0

Windows Subsystem for Linux (WSL)

# Install WSL 2
wsl --install

# List available distributions
wsl --list --online

# Install specific distribution
wsl --install -d Ubuntu-22.04

# Set default WSL version
wsl --set-default-version 2

# List installed distributions
wsl --list --verbose

# Run command in WSL
wsl ls -la

# Access Windows files from WSL
cd /mnt/c/Users/YourName/Documents

# Access WSL files from Windows
\\wsl$\Ubuntu-22.04\home\username\

Windows Package Managers

# Winget (Windows Package Manager)
# Search for package
winget search vscode

# Install package
winget install Microsoft.VisualStudioCode

# Upgrade all packages
winget upgrade --all

# List installed packages
winget list

# Chocolatey
# Install Chocolatey
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

# Install package with Chocolatey
choco install googlechrome
choco install nodejs
choco install git
