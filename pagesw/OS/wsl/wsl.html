🐧 WSL (Windows Subsystem for Linux)

Run a GNU/Linux environment directly on Windows without the overhead of a virtual machine.

Overview

Windows Linux Development Microsoft

WSL lets developers run a GNU/Linux environment including most command-line tools, utilities, and applications directly on Windows, unmodified, without the overhead of a traditional virtual machine or dual-boot setup.

Key Features

  • Native Linux — Real Linux kernel (WSL 2)
  • Performance — Fast file system and full system call compatibility
  • Integration — Access Windows files from Linux and vice versa
  • Multiple Distros — Ubuntu, Debian, Kali, openSUSE, etc.
  • GUI Support — Run Linux GUI applications (WSLg)
  • GPU Acceleration — CUDA and DirectML support
  • Docker Integration — Docker Desktop with WSL 2 backend
  • VS Code Integration — Remote development in WSL

Common Use Cases

  • Web development with Linux tools
  • Running Linux command-line utilities
  • Docker and Kubernetes development
  • DevOps and CI/CD workflows
  • Data science and ML with GPU support
  • Cross-platform development
  • Learning Linux without dual-boot
  • Security testing with Kali Linux

Getting Started

Install WSL (PowerShell as Admin):

# Install WSL with Ubuntu (default)
wsl --install

# Install specific distro
wsl --install -d Ubuntu-22.04

# List available distros
wsl --list --online

# Set WSL 2 as default
wsl --set-default-version 2

# Update WSL
wsl --update

Basic usage:

# Launch default Linux distribution
wsl

# Run specific command
wsl ls -la

# Access Windows files from Linux
cd /mnt/c/Users/YourName

# Access Linux files from Windows
\\wsl$\Ubuntu\home\username

# VS Code in WSL
code .

Best Practices

  • Use WSL 2 for better performance
  • Store project files in Linux filesystem for speed
  • Use VS Code Remote - WSL extension
  • Set up Docker Desktop with WSL 2 backend
  • Configure Git in WSL separately from Windows
  • Use Windows Terminal for better experience
  • Back up your WSL distributions regularly
  • Leverage WSLg for GUI applications
