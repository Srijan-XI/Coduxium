Overview

CLOUD AI/ML BIG DATA KUBERNETES

Google Cloud Platform (GCP) is a suite of cloud computing services running on Google's infrastructure. It excels in big data processing, machine learning, and containerized applications with Kubernetes.

Core Services

  • Compute Engine — Virtual machines on Google infrastructure
  • App Engine — Fully managed serverless platform
  • Cloud Functions — Event-driven serverless compute
  • Cloud Storage — Object storage with global edge caching
  • Cloud SQL — Managed MySQL, PostgreSQL, SQL Server
  • Firestore/Datastore — NoSQL document database
  • GKE (Google Kubernetes Engine) — Managed Kubernetes
  • BigQuery — Serverless data warehouse

gcloud CLI Installation

# Linux/macOS
curl https://sdk.cloud.google.com | bash
exec -l $SHELL

# Windows (using installer)
# Download from: https://cloud.google.com/sdk/docs/install

# Initialize gcloud
gcloud init

# Authenticate
gcloud auth login

# Set default project
gcloud config set project PROJECT_ID

# List projects
gcloud projects list

# Verify installation
gcloud --version

Compute Engine Operations

# Create VM instance
gcloud compute instances create my-instance \
    --zone=us-central1-a \
    --machine-type=e2-medium \
    --image-family=ubuntu-2204-lts \
    --image-project=ubuntu-os-cloud \
    --boot-disk-size=10GB

# List instances
gcloud compute instances list

# Start instance
gcloud compute instances start my-instance --zone=us-central1-a

# Stop instance
gcloud compute instances stop my-instance --zone=us-central1-a

# SSH into instance
gcloud compute ssh my-instance --zone=us-central1-a

# Delete instance
gcloud compute instances delete my-instance --zone=us-central1-a

# Get instance details
gcloud compute instances describe my-instance --zone=us-central1-a

gsutil rsync -r ./local-folder gs://my-bucket-name/folder/

# List objects in bucket
gsutil ls gs://my-bucket-name/

# Delete bucket
gsutil rb gs://my-bucket-name

# Make object public
gsutil acl ch -u AllUsers:R gs://my-bucket-name/file.txt

Cloud Functions Example

# Node.js HTTP Function
exports.helloWorld = (req, res) => {
    const name = req.query.name || req.body.name || 'World';
    res.send(`Hello ${name}!`);
};

# Python HTTP Function
def hello_world(request):
    request_json = request.get_json(silent=True)
    request_args = request.args

    if request_json and 'name' in request_json:
        name = request_json['name']
    elif request_args and 'name' in request_args:
        name = request_args['name']
    else:
        name = 'World'
    
    return f'Hello {name}!'

# Deploy function
gcloud functions deploy helloWorld \
    --runtime nodejs18 \
    --trigger-http \
    --allow-unauthenticated \
    --entry-point helloWorld

# Invoke function
curl https://REGION-PROJECT_ID.cloudfunctions.net/helloWorld?name=User

GCP SDK Example (Python)

from google.cloud import storage
from google.cloud import compute_v1

# Cloud Storage operations
storage_client = storage.Client()

# Upload blob
bucket = storage_client.bucket('my-bucket')
blob = bucket.blob('myfile.txt')
blob.upload_from_filename('local-file.txt')

# Download blob
blob.download_to_filename('downloaded-file.txt')

# List blobs
blobs = storage_client.list_blobs('my-bucket')
for blob in blobs:
    print(blob.name)

# Compute Engine operations
instance_client = compute_v1.InstancesClient()

# List instances
request = compute_v1.AggregatedListInstancesRequest()
request.project = "your-project-id"

agg_list = instance_client.aggregated_list(request=request)
for zone, response in agg_list:
    if response.instances:
        for instance in response.instances:
            print(f"Instance: {instance.name}, Zone: {zone}")

# BigQuery operations
from google.cloud import bigquery

client = bigquery.Client()

query = """
    SELECT name, COUNT(*) as count
    FROM `bigquery-public-data.usa_names.usa_1910_current`
    WHERE state = 'TX'
    GROUP BY name
    ORDER BY count DESC
    LIMIT 10
"""

query_job = client.query(query)
for row in query_job:
    print(f"{row.name}: {row.count}")

Deployment Manager Template

# deployment.yaml
resources:
- name: my-vm
  type: compute.v1.instance
  properties:
    zone: us-central1-a
    machineType: zones/us-central1-a/machineTypes/e2-medium
    disks:
    - deviceName: boot
      type: PERSISTENT
      boot: true
      autoDelete: true
      initializeParams:
        sourceImage: projects/ubuntu-os-cloud/global/images/family/ubuntu-2204-lts
    networkInterfaces:
    - network: global/networks/default
      accessConfigs:
      - name: External NAT
        type: ONE_TO_ONE_NAT

- name: my-bucket
  type: storage.v1.bucket
  properties:
    location: US
    storageClass: STANDARD

# Deploy
gcloud deployment-manager deployments create my-deployment \
    --config deployment.yaml

# Delete deployment
gcloud deployment-manager deployments delete my-deployment
